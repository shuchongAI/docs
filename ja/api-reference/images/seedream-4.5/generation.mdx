<ParamField header="Authorization" type="string" required>

  [APIキー管理ページ](https://apimart.ai/console/token) にアクセスしてAPIキーを取得してください

  リクエストヘッダーに追加:

## Body

<ParamField body="model" type="string" default="doubao-seedance-4-5" required>
  画像生成モデル名

  Example: `"doubao-seedance-4-5"`
</ParamField>

<ParamField body="prompt" type="string" required>
  画像生成のためのテキスト説明
</ParamField>

<ParamField body="size" type="string" default="1:1">
  画像のアスペクト比

  サポートされているアスペクト比:
  - `1:1` - 正方形（デフォルト）
  - `4:3` - 横向き 4:3
  - `3:4` - 縦向き 3:4
  - `16:9` - 横向きワイドスクリーン
  - `9:16` - 縦向き
  - `3:2` - 横向き 3:2
  - `2:3` - 縦向き 2:3
  - `21:9` - ウルトラワイド
  - `9:21` - ウルトラトール
  - `auto` - 参照画像のアスペクト比に自動マッチ（image_urls が必要）
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  画像解像度

  サポートされている解像度:
  - `2K` - 標準解像度（デフォルト）
  - `4K` - 高解像度

  > **注意：** Seedream-4.5 は 1K 解像度をサポートしていません

  **解像度参考サイズ：**
  | 解像度 | 1:1 サイズ | 16:9 サイズ |
  |--------|----------|-----------|
  | 2K | 2048x2048 | 2560x1440 |
  | 4K | 4096x4096 | 5404x3040 |

  <Expandable title="完全なピクセルマッピング表">
    **2K 解像度**
    | 比率 | ピクセル |
    |------|--------|
    | 1:1 | 2048x2048 |
    | 4:3 | 2304x1728 |
    | 3:4 | 1728x2304 |
    | 16:9 | 2560x1440 |
    | 9:16 | 1440x2560 |
    | 3:2 | 2496x1664 |
    | 2:3 | 1664x2496 |
    | 21:9 | 3024x1296 |
    | 9:21 | 1296x3024 |

    **4K 解像度**
    | 比率 | ピクセル |
    |------|--------|
    | 1:1 | 4096x4096 |
    | 4:3 | 4694x3520 |
    | 3:4 | 3520x4694 |
    | 16:9 | 5404x3040 |
    | 9:16 | 3040x5404 |
    | 3:2 | 4992x3328 |
    | 2:3 | 3328x4992 |
    | 21:9 | 6198x2656 |
    | 9:21 | 2656x6198 |
  </Expandable>
</ParamField>

<ParamField body="n" type="integer">
  生成する画像の数

  範囲：1-15（最小 1 枚、最大 15 枚）
  
  デフォルト：1

  **注意：** 
  - 実際に生成できる画像の数は、このパラメータと入力参照画像の数の両方に影響されます。参照画像の数 + 生成画像の数 ≤ 15 枚
  - **純粋な数字を入力してください（例：`1`）。引用符を使用するとエラーになります**

  枚数に基づいて料金が事前に差し引かれます
</ParamField>

<ParamField body="image_urls" type="array">
  画像から画像への変換または画像編集のための参照画像URLリスト

  - 最大 10 枚の画像
  - 画像形式：jpeg、png
  - アスペクト比（幅/高さ）範囲：[1/3, 3]
  - 幅と高さ（px）> 14
  - サイズ：10MB以下
  - 総ピクセル数：6000×6000 px以下
</ParamField>

<ParamField body="optimize_prompt_options.mode" type="string" default="standard">
  プロンプト最適化モード

  - `standard`：標準モード、より高品質なコンテンツを生成しますが、処理時間が長くなります
  - `fast`：高速モード、処理時間が短く、通常の品質でコンテンツを生成します

  デフォルト値：`standard`
</ParamField>

<ParamField body="sequential_image_generation" type="string" default="disabled">
  連続画像生成モード（Doubao固有の機能）

  複数の画像を生成するかどうかを制御します：

  - `disabled`：連続モード無効、複数の参照画像があっても1枚のみ生成（デフォルト）
  - `auto`：連続モード有効、複数の画像を生成可能

  **使用方法：**
  - ✅ `image_urls` を提供する必要があります - 少なくとも1枚の参照画像が必要
  - ✅ `n: 3` を設定するか、`sequential_image_generation: "auto"` + `max_images: 3` を使用
  - ✅ これにより参照画像に基づいて3枚の異なる画像が生成されます
  - ⚠️ `n > 1` の場合、自動的に `auto` に設定されます

  **制限：**
  - 純粋なテキストから画像への変換（`image_urls` なし）では複数の画像を生成できません - これはDoubao APIの制限です
</ParamField>

<ParamField body="sequential_image_generation_options" type="object">
  連続画像生成オプション

  `sequential_image_generation` が `auto` に設定されている場合に使用可能

  **プロパティ：**
  - `max_images` (integer): 生成する画像の数を指定、範囲：1-15

  **例：**
  ```json
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 3
  }
  ```
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  生成された画像に透かしを追加するかどうか

  - `true`：透かしを追加
  - `false`：透かしなし（デフォルト）
</ParamField>

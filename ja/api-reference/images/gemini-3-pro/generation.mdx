## Authorizations

<ParamField header="Authorization" type="string" required>
  すべてのAPIはBearer Tokenによる認証が必要です

  APIキーの取得：

  [APIキー管理ページ](https://apimart.ai/console/token)にアクセスしてAPIキーを取得してください

  リクエストヘッダーに追加：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" default="gemini-3-pro-image-preview" required>
  画像生成モデル名

  例: `"gemini-3-pro-image-preview"`
</ParamField>

<ParamField body="prompt" type="string" required>
  画像生成のテキスト説明
</ParamField>

<ParamField body="size" type="string">
  画像生成のサイズ

  サポートされている形式：
  - 比率：`1:1`、`2:3`、`3:2`、`3:4`、`4:3`、`4:5`、`5:4`、`9:16`、`16:9`、`21:9`
</ParamField>

<ParamField body="n" type="integer">
  生成する画像の数

  1に固定
  
  **⚠️ 注意：** 純粋な数字（例：`1`）を入力してください。引用符を付けないでください。エラーになります
</ParamField>

<ParamField body="resolution" type="string">
  出力画像の解像度
  
  サポートされる値：
  - `1K` - 1K解像度（デフォルト）
  - `2K` - 2K解像度
  - `4K` - 4K解像度
  
  **⚠️ 注意：** base64形式で4K画像を生成する場合、処理時間が長くなります
</ParamField>

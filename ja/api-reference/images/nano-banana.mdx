# Nano-Banana Image Generation

Use the Nano-Banana model for fast image generation.

## Highlights

- Asynchronous processing: returns a task ID for subsequent queries
- Fast generation: faster than other models in many cases
- Expiring links: generated image links are valid for 24 hours

## Request example

<RequestExample>

```bash
curl -X POST https://apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana",
    "prompt": "Modern city night scene with neon lights",
    "size": "1024x1024",
    "n": 1
  }'
```

</RequestExample>

## Parameters

<ParamField query="model" type="string" required>
  Model name. Use "nano-banana".
</ParamField>

<ParamField query="prompt" type="string" required>
  Text description for image generation.
</ParamField>

<ParamField query="size" type="string">
  Output size. Supported: "1024x1024", "1792x1024", "1024x1792".
</ParamField>

<ParamField query="n" type="integer">
  Number of images to generate. Default: 1.
</ParamField>

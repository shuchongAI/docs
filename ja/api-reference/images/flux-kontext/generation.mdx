## サポートされているモデル

| モデル名 | 説明 | 課金方式 |
|----------|------|----------|
| `flux-kontext-pro` | Flux Kontext Pro 画像編集モデル | 固定価格 |
| `flux-kontext-max` | Flux Kontext Max 画像編集モデル（より高品質） | 固定価格 |

## 認証

<ParamField header="Authorization" type="string" required>
  すべてのエンドポイントはBearer Token認証が必要です

  API Keyの取得：

  [API Key管理ページ](https://apimart.ai/console/token)でAPI Keyを取得してください

  リクエストヘッダーに追加：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Body

<ParamField body="model" type="string" required>
  モデル名

  - `flux-kontext-pro` - Kontext Proモデル
  - `flux-kontext-max` - Kontext Maxモデル（より高品質）
</ParamField>

<ParamField body="prompt" type="string" required>
  画像生成のテキスト説明
</ParamField>

<ParamField body="image_urls" type="array">
  入力画像URLリスト

  **制限：**
  - 1枚のみサポート
  - 公開アクセス可能なURLである必要があります
  - Base64形式はサポートされていません
</ParamField>

<ParamField body="size" type="string" default="画像あり: match_input_image、なし: 16:9">
  画像のアスペクト比

  サポートされているアスペクト比:
  - `match_input_image` - 入力画像の比率に合わせる（画像提供時のデフォルト）
  - `auto` - `match_input_image`と同じ
  - `1:1` - 正方形
  - `4:3` - 横向き 4:3
  - `3:4` - 縦向き 3:4
  - `16:9` - 横向きワイドスクリーン（画像なしの場合のデフォルト）
  - `9:16` - 縦向き
  - `3:2` - 横向き 3:2
  - `2:3` - 縦向き 2:3
  - `21:9` - ウルトラワイド
  - `9:21` - ウルトラトール
</ParamField>

<ParamField body="response_format" type="string" default="png">
  出力画像形式

  - `png` - PNG形式（デフォルト）
  - `jpg` - JPEG形式
</ParamField>

<ParamField body="safety_tolerance" type="integer" default="2">
  安全許容度

  範囲：0-6、値が高いほど寛容
</ParamField>

<ParamField body="prompt_upsampling" type="boolean" default="false">
  プロンプト強化を有効にする

  - `true` - 有効
  - `false` - 無効（デフォルト）
</ParamField>

## 使用例

**画像編集（入力画像あり）**
```json
{
    "model": "flux-kontext-pro",
    "prompt": "背景をビーチに変更",
    "image_urls": ["https://example.com/photo.jpg"]
}
```

**テキストから画像（入力画像なし）**
```json
{
    "model": "flux-kontext-pro",
    "prompt": "青い猫",
    "size": "16:9"
}
```

## 注意事項

1. **画像URL要件**：入力画像は公開アクセス可能なURLである必要があります、base64はサポートされていません
2. **結果保存**：生成された画像は自動的に保存され、URLは24時間有効です
3. **タスクポーリング**：タスクは非同期で処理されます、結果を取得するには `/v1/tasks/{task_id}` をポーリングしてください
4. **入力画像制限**：1枚のみサポート

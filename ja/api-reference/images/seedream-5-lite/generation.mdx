```java Java
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String url = "https://api.apimart.ai/v1/images/generations";

        String payload = """
        {
          "model": "doubao-seedream-5-0-lite",
          "prompt": "庭で遊ぶゴールデンレトリバー、晴れた日、高解像度写真スタイル",
          "size": "16:9",
          "resolution": "2K",
          "n": 1
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(url))
            .header("Authorization", "Bearer <token>")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());

        System.out.println(response.body());
    }
}
```

## Body

<ParamField body="model" type="string" default="doubao-seedream-5-0-lite" required>
  画像生成モデル名

  Example: `"doubao-seedream-5-0-lite"`
</ParamField>

<ParamField body="prompt" type="string" required>
  画像生成のテキスト説明
</ParamField>

<ParamField body="size" type="string" default="1:1">
  画像のアスペクト比

  サポートされるアスペクト比：
  - `1:1` - 正方形（デフォルト）
  - `4:3` - 横向き 4:3
  - `3:4` - 縦向き 3:4
  - `16:9` - 横向きワイドスクリーン
  - `9:16` - 縦向き
  - `3:2` - 横向き 3:2
  - `2:3` - 縦向き 2:3
  - `21:9` - ウルトラワイド
  - `auto` - 参照画像のアスペクト比に自動マッチ（image_urlsが必要）

  > **注意：** Seedream-5.0-Liteは `9:21` 比率をサポートしていません
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  画像解像度

  サポートされる解像度：
  - `2K` - 標準解像度（デフォルト）
  - `3K` - 高解像度

  > **注意：** Seedream-5.0-Liteは2Kと3K解像度のみサポートしています。1Kと4Kはサポートされていません

  **解像度参考サイズ：**
  | 解像度 | 1:1 サイズ | 16:9 サイズ |
  |--------|-----------|------------|
  | 2K | 2048x2048 | 2848x1600 |
  | 3K | 3072x3072 | 4096x2304 |

  <Expandable title="完全なピクセルマッピング表">
    **2K 解像度**
    | 比率 | ピクセル |
    |------|---------|
    | 1:1 | 2048×2048 |
    | 4:3 | 2304×1728 |
    | 3:4 | 1728×2304 |
    | 16:9 | 2848×1600 |
    | 9:16 | 1600×2848 |
    | 3:2 | 2496×1664 |
    | 2:3 | 1664×2496 |
    | 21:9 | 3136×1344 |

    **3K 解像度**
    | 比率 | ピクセル |
    |------|---------|
    | 1:1 | 3072×3072 |
    | 4:3 | 3456×2592 |
    | 3:4 | 2592×3456 |
    | 16:9 | 4096×2304 |
    | 9:16 | 2304×4096 |
    | 3:2 | 3744×2496 |
    | 2:3 | 2496×3744 |
    | 21:9 | 4704×2016 |
  </Expandable>
</ParamField>

<ParamField body="n" type="integer">
  生成する画像の数

  範囲：1-4

  デフォルト：1

  **注意：**
  - `n > 1` の場合、連続画像生成モードが自動的に有効になります
  - **数値のみ入力してください（例：`1`）。引用符を使用するとエラーが発生します**

  生成数に基づいて事前に課金されます
</ParamField>

<ParamField body="image_urls" type="array">
  画像から画像生成用の参照画像URLリスト

  **制限：**
  - 画像形式：jpeg、png
  - アスペクト比（幅/高さ）範囲：[1/3, 3]
  - 幅と高さ（px）> 14
  - サイズ：画像あたり10MB以下
  - 総ピクセル数：6000×6000 px以下
</ParamField>

<ParamField body="output_format" type="string" default="jpeg">
  出力画像フォーマット

  - `jpeg`：JPEGフォーマット（デフォルト）
  - `png`：PNGフォーマット、透明背景などに適しています

  > **注意：** `output_format` パラメータはSeedream-5.0-Lite専用です。他の画像モデルではこのパラメータは無視されます。
</ParamField>

<ParamField body="sequential_image_generation" type="string" default="disabled">
  連続画像生成モード

  複数の画像を生成するかどうかを制御します：

  - `disabled`：連続モードを無効にする（デフォルト）
  - `auto`：連続モードを有効にし、複数の画像を生成できます

  **使用上の注意：**
  - `n > 1` の場合、自動的に `auto` に設定されます
</ParamField>

<ParamField body="sequential_image_generation_options" type="object">
  連続画像生成オプション

  `sequential_image_generation` が `auto` に設定されている場合に使用可能

  **プロパティ：**
  - `max_images` (integer): 生成する画像の数を指定

  **例：**
  ```json
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 4
  }
  ```
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  生成された画像にウォーターマークを追加するかどうか

  - `true`：ウォーターマークを追加
  - `false`：ウォーターマークなし（デフォルト）
</ParamField>

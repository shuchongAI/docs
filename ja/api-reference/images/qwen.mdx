# Qwen 画像生成

高品質な画像生成のためにQwenモデルを使用します。

## ハイライト

- 非同期処理: 後続のクエリ用にタスクIDを返します
- 複数のモード: テキストから画像、画像から画像、画像編集
- リンクの有効期限: 生成された画像のリンクは24時間有効です

## リクエスト例

<RequestExample>

```bash
curl -X POST https://apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "qwen",
    "prompt": "古典的な中国庭園の蓮の花",
    "size": "1024x1024",
    "n": 1
  }'
```

</RequestExample>

## パラメータ

<ParamField query="model" type="string" required>
  モデル名。"qwen"を使用してください。
</ParamField>

<ParamField query="prompt" type="string" required>
  画像生成のためのテキスト説明。
</ParamField>

<ParamField query="size" type="string">
  出力サイズ。サポート: "1024x1024", "1792x1024", "1024x1792"。
</ParamField>

<ParamField query="n" type="integer">
  生成する画像の数。デフォルト: 1。
</ParamField>

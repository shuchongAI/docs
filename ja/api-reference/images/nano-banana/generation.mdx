## 認証

<ParamField header="Authorization" type="string" required>
  すべてのAPIエンドポイントはBearer Token認証が必要です

  APIキーの取得方法：

  [APIキー管理ページ](https://api.apimart.ai/console/token)にアクセスしてAPIキーを取得してください

  リクエストヘッダーに追加：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## リクエストボディ

<ParamField body="model" type="string" default="gemini-2.5-flash-image-preview" required>
  画像生成モデル名

  Example: `"gemini-2.5-flash-image-preview"`
</ParamField>

<ParamField body="prompt" type="string" required>
  画像生成のためのテキスト説明

  最大1000文字
</ParamField>

<ParamField body="size" type="string">
  画像生成サイズ

  サポートされている形式：
  - 比率：`1:1`、`2:3`、`3:2`、`3:4`、`4:3`、`4:5`、`5:4`、`9:16`、`16:9`、`21:9`
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成する画像の数

  1に固定
  
  **⚠️ 注意：** 純粋な数字を入力してください（例：`1`）。引用符は使用しないでください
</ParamField>

<ParamField body="image_urls" type="array">
  画像から画像または画像編集のための参照画像URLリスト

  - 最大5枚の画像
  - 各画像は10MBを超えないこと
  - サポート形式：.jpeg, .jpg, .png, .webp
</ParamField>

<ParamField body="mask_url" type="string">
  マスク画像URL

  - PNG形式である必要があります
  - サイズは参照画像と一致する必要があります
  - 最大4MB
</ParamField>

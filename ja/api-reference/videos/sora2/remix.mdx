## パスパラメータ

<ParamField path="video_id" type="string" required>
  元のビデオタスクID

  これは以前のビデオ生成リクエストから返されたタスクIDです
</ParamField>

## 認証

<ParamField header="Authorization" type="string" required>
  すべてのAPIエンドポイントはBearerトークン認証が必要です

  APIキーを取得：

  [APIキー管理ページ](https://api.apimart.ai/console/token)にアクセスしてAPIキーを取得してください

  リクエストヘッダーに追加：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## ボディ

<ParamField body="model" type="string" default="sora-2" required>
  ビデオリミックスモデル名

  サポートされているモデル：
  - `sora-2` - 標準バージョン
  - `sora-2-pro` - プロフェッショナルバージョン、より長い時間をサポート

  例：`"sora-2"` または `"sora-2-pro"`
</ParamField>

<ParamField body="prompt" type="string" required>
  リミックス指示の説明
</ParamField>

<ParamField body="duration" type="integer">
  ビデオの長さ（秒）

  - `sora-2`：10秒または15秒をサポート
  - `sora-2-pro`：15秒（HD）または25秒をサポート

  例：`15`
</ParamField>

<ParamField body="aspect_ratio" type="string">
  ビデオのアスペクト比

  サポートされている比率：
  - `16:9` - 標準ワイドスクリーン（推奨）
  - `9:16` - 縦向きモード
  - `1:1` - 正方形

  例：`"16:9"`
</ParamField>

## レスポンス

<ResponseField name="created" type="integer">
  タスク作成タイムスタンプ
</ResponseField>

<ResponseField name="id" type="string">
  一意のタスク識別子
</ResponseField>

<ResponseField name="model" type="string">
  実際に使用されたモデル名
</ResponseField>

<ResponseField name="object" type="string">
  オブジェクトタイプ、`video.remix.task`に固定
</ResponseField>

<ResponseField name="progress" type="integer">
  タスク完了進捗率 (0-100)
</ResponseField>

<ResponseField name="status" type="string">
  タスクステータス

  可能な値：
  - `pending` - 処理待ち
  - `processing` - 処理中
  - `completed` - 完了
  - `failed` - 失敗
</ResponseField>

<ResponseField name="task_info" type="object">
  タスク詳細

  <Expandable title="プロパティ">

    <ResponseField name="estimated_time" type="integer">

    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="type" type="string">
  出力タイプ、`video`に固定
</ResponseField>

<ResponseField name="usage" type="object">

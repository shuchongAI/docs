# Wan2.5-Video 生成

高品質な動画生成にWan2.5-Videoモデルを使用します。

## ハイライト

- 非同期処理: 後続のクエリのためのタスクIDを返します
- 複数モード: テキストから動画、画像から動画
- 有効期限付きリンク: 生成された動画リンクは24時間有効です

## リクエスト例

<RequestExample>

```bash
curl -X POST https://apimart.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.5-video",
    "prompt": "夜の都市交通",
    "duration": 12,
    "resolution": "1920x1080"
  }'
```

</RequestExample>

## パラメータ

<ParamField query="model" type="string" required>
  モデル名。"wan2.5-video"を使用します。
</ParamField>

<ParamField query="prompt" type="string" required>
  動画生成のためのテキスト説明。
</ParamField>

<ParamField query="duration" type="integer">
  動画の長さ（秒）。デフォルト: 12。
</ParamField>

<ParamField query="resolution" type="string">
  動画の解像度。対応: "1920x1080"、"1280x720"。
</ParamField>

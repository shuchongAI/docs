title: "Sora2 キャラクタークエリ（キャラクター管理）"
description: "
- キャラクター作成タスクのステータスと結果をクエリ

- 作成されたキャラクターの詳細情報を取得

- 後続の動画生成に使用できるキャラクターIDを返す
"
api: "GET https://api.apimart.ai/v1/characters_tasks/{task_id}"

---




<RequestExample>

```bash cURL
curl --request GET \
  --url https://api.apimart.ai/v1/characters_tasks/task_01KBZ0TZZ7C7M3WS1TJM0PH8J4 \
  --header 'Authorization: Bearer <token>'
```

```python Python
import requests

task_id = "task_01KBZ0TZZ7C7M3WS1TJM0PH8J4"
url = f"https://api.apimart.ai/v1/characters_tasks/{task_id}"

headers = {
    "Authorization": "Bearer <token>"
}

response = requests.get(url, headers=headers)

print(response.json())
```

```javascript JavaScript
const taskId = "task_01KBZ0TZZ7C7M3WS1TJM0PH8J4";
const url = `https://api.apimart.ai/v1/characters_tasks/${taskId}`;

const headers = {
  "Authorization": "Bearer <token>"
};

fetch(url, {
  method: "GET",
  headers: headers
})
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));
```

## パスパラメータ

<ParamField path="task_id" type="string" required>
  キャラクター作成タスクID

  キャラクター作成APIから返されたタスクID

  例：`"task_01KBZ0TZZ7C7M3WS1TJM0PH8J4"`
</ParamField>

## 認証

<ParamField header="Authorization" type="string" required>
  すべてのAPIエンドポイントはBearer Token認証が必要です

  APIキーの取得：

  [APIキー管理ページ](https://apimart.ai/console/token)でAPIキーを取得してください

  リクエストヘッダーに追加：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## Response

<ResponseField name="code" type="integer">
  レスポンスステータスコード、成功時は200
</ResponseField>

<ResponseField name="data" type="object">
  レスポンスデータオブジェクト

  <Expandable title="フィールド詳細">
    <ResponseField name="id" type="string">
      タスクの一意識別子

      例：`"task_01KC0JZCMTMQ70D68XTM56Q5D0"`
    </ResponseField>

    <ResponseField name="status" type="string">
      タスクステータス

      可能な値：
      - `pending` - 処理待ち
      - `processing` - 処理中
      - `completed` - 完了
      - `failed` - 失敗

      例：`"completed"`
    </ResponseField>

    <ResponseField name="progress" type="integer">
      タスク完了進捗率 (0-100)

      例：`100`
    </ResponseField>

    <ResponseField name="created" type="integer">
      タスク作成タイムスタンプ（Unixタイムスタンプ）

      例：`1765251461`
    </ResponseField>

    <ResponseField name="completed" type="integer">
      タスク完了タイムスタンプ（Unixタイムスタンプ）

      例：`1765251507`
    </ResponseField>

    <ResponseField name="estimated_time" type="integer">
      推定処理時間（秒）

      例：`30`
    </ResponseField>

    <ResponseField name="actual_time" type="integer">
      実際の処理時間（秒）

      例：`46`
    </ResponseField>

    <ResponseField name="result" type="object">
      タスク結果

      <Expandable title="resultフィールド">
        <ResponseField name="characters" type="array">
          キャラクター配列

          <Expandable title="キャラクターオブジェクト">
            <ResponseField name="id" type="string">
              キャラクターの一意識別子
              
              このIDは動画生成時に `character_url` パラメータでキャラクターを参照するために使用できます

              例：`"ch_6937998961208191a45ef08447a554df"`
            </ResponseField>

            <ResponseField name="display_name" type="string">
              キャラクターの表示名

              例：`"Turbo Whiskers"`
            </ResponseField>

            <ResponseField name="profile_picture_url" type="string">
              キャラクターのプロフィール画像URL

              例：`"https://upload.apimart.ai/f/image/...character_task_xxx.jpg"`
            </ResponseField>

            <ResponseField name="username" type="string">
              キャラクターのユーザー名、プロンプトで `@username` を使用して参照可能

              例：`"duksvfkf.turbo_whis"`
            </ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

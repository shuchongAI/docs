## 認証

<ParamField header="Authorization" type="string" required>
  すべてのAPIエンドポイントにはBearer Token認証が必要です

  APIキーの取得:

  [APIキー管理ページ](https://api.apimart.ai/console/token)にアクセスしてAPIキーを取得してください

  リクエストヘッダーに追加:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## ボディ

<ParamField body="model" type="string" default="sora-2" required>
  動画生成モデル名

  サポートされているモデル：
  - `sora-2` - 標準バージョン
  - `sora-2-pro` - プロフェッショナルバージョン、より長い時間をサポート

  例：`"sora-2"` または `"sora-2-pro"`
</ParamField>

<ParamField body="prompt" type="string" required>
  動画生成用のテキスト記述

  最大1000文字
</ParamField>

<ParamField body="duration" type="integer">
  動画の長さ（秒）

  - `sora-2`：10秒または15秒をサポート
  - `sora-2-pro`：15秒（HD）または25秒をサポート

  例：`10`
</ParamField>

<ParamField body="aspect_ratio" type="string">
  動画の解像度

  対応フォーマット:
  - `16:9` (横向き)
  - `9:16` (縦向き)
</ParamField>

<ParamField body="image_urls" type="array">
  画像から動画生成のための参照画像URL配列

  **💡 クイック入力（Try itエリア）：**
  1. "+ Add an item" をクリックして画像URLを追加
  2. 画像の完全なURLアドレスを入力

  <Expandable title="フィールド詳細">
    <ParamField body="url" type="string" required>
      画像URLアドレス
      
      以下が可能：
      - 公開アクセス可能な画像URL（http:// または https://）
      - Base64エンコードされた画像データ
      
      **制限：**
      - 最大ファイルサイズ：10MB
      - 対応フォーマット：.jpeg、.jpg、.png、.webp
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  生成された動画にウォーターマークを追加するかどうか

  - `false`：ウォーターマークを追加しない
  - `true`：動画にSora公式ウォーターマークを追加

  デフォルト: `false`
</ParamField>

<ParamField body="thumbnail" type="boolean">
  動画のサムネイルを生成するかどうか

  - `true`：プレビューと表示用の動画サムネイルを生成
  
  サムネイルが不要な場合は、このパラメータを省略できます
</ParamField>

<ParamField body="private" type="boolean" default="false">
  プライバシーモードを有効にするかどうか

  - `true`：動画は公開されず、remix操作（再編集）もできません
  - `false`：動画を公開およびremixできます

  デフォルト: `false`
</ParamField>

## レスポンス

<ResponseField name="code" type="integer">
  レスポンスステータスコード、成功時は200
</ResponseField>

<ResponseField name="data" type="array">

    <ResponseField name="task_id" type="string">

    </ResponseField>

## 認証

<ParamField header="Authorization" type="string" required>
  すべてのAPIエンドポイントでBearer Tokenによる認証が必要です

  API Keyの取得：

  [API Key管理ページ](https://api.apimart.ai/console/token)にアクセスしてAPI Keyを取得してください

  使用時はリクエストヘッダーに追加：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## リクエストボディ

<ParamField body="model" type="string" default="veo3.1" required>
  動画生成モデル名

  Example: `"veo3.1"`
</ParamField>

<ParamField body="prompt" type="string" required>
  動画生成のテキスト説明

  最大1000文字
</ParamField>

<ParamField body="duration" type="integer">
  動画の長さ（秒）

  固定値：`8`（VEO3 は 8 秒の長さのみサポート）
</ParamField>

<ParamField body="aspect_ratio" type="string">
  動画のアスペクト比

  サポートされている形式：
  - `16:9` (横向き)
  - `9:16` (縦向き)
</ParamField>

<ParamField body="image_urls" type="array">
  動画生成に使用する参考画像URLの配列

  - 各要素は有効な画像URLである必要があります
  - サポート形式：.jpeg、.jpg、.png、.webp
  - 最大ファイルサイズ：10MB
</ParamField>

## レスポンス

<ResponseField name="created" type="integer">
  タスク作成時のタイムスタンプ
</ResponseField>

<ResponseField name="id" type="string">
  タスクの一意識別子
</ResponseField>

<ResponseField name="model" type="string">
  実際に使用されたモデル名
</ResponseField>

<ResponseField name="object" type="string">
  オブジェクトタイプ、`video.generation.task`に固定
</ResponseField>

<ResponseField name="progress" type="integer">
  タスク完了進捗率（0-100）
</ResponseField>

<ResponseField name="status" type="string">
  タスクの状態

  可能な値：
  - `pending` - 処理待ち
  - `processing` - 処理中
  - `completed` - 完了
  - `failed` - 失敗
</ResponseField>

<ResponseField name="task_info" type="object">
  タスクの詳細情報

  <Expandable title="属性">
    <ResponseField name="can_cancel" type="boolean">
      タスクをキャンセルできるかどうか
    </ResponseField>

    <ResponseField name="estimated_time" type="integer">
      予想完了時間（秒）
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="type" type="string">
  出力タイプ、`video`に固定
</ResponseField>

<ResponseField name="usage" type="object">
  課金情報

  <Expandable title="属性">
    <ResponseField name="total_tokens" type="integer">
      消費された総トークン数
    </ResponseField>
  </Expandable>
</ResponseField>

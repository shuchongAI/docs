## プラットフォームの利点

サブスクリプション不要の従量課金。全モデル明確なトークン単価、公式プロバイダーより手頃な料金で、同じ品質と低レイテンシを維持。

- 全モデル明確なトークン単価
- エンタープライズ利用でボリューム割引
- 隠れた費用なし
- 実際の使用量のみ課金

### マルチプロバイダールーティングとエンタープライズ SLA

複数の LLM プロバイダー間でインテリジェントルーティングを内蔵し、高可用性と低レイテンシを確保。あるプロバイダーに問題が発生しても、リクエストは自動的にバックアッププロバイダーにルーティングされ、サービス中断なし。

### 99.9% 可用性 SLA

自動フェイルオーバー付きのエンタープライズグレードの信頼性。マルチプロバイダールーティングで最小レイテンシでのリクエスト処理を保証。

### グローバル CDN 高速化

世界中のエッジロケーションで最低レイテンシを実現。最適化されたルーティングで応答時間を短縮し、グローバルなユーザー体験を向上。

### レート制限管理

プロバイダー間で自動的にレート制限を処理。インテリジェントなリクエスト分散でスロットリングを防止し、スムーズな運用を保証。

### リアルタイムステータス

エンドポイントの健全性とパフォーマンスを監視。画像・動画生成の非同期タスク進行状況を追跡、Webhook 通知サポート。[ステータス確認 →](/ja/api-reference/tasks/status)

## SDK とコード例

Python、Node.js、Java 用の公式 SDK。すべての SDK が OpenAI 互換で、Base URL 設定の変更のみでシームレスに移行可能。

### Python SDK

`pip install openai` でインストール。Base URL を設定し、既存の OpenAI コードをそのまま使用、変更不要。

```python
from openai import OpenAI
client = OpenAI(
    base_url="https://api.apimart.ai/v1",
    api_key="your-apimart-key"
)
```

### Node.js SDK

`npm install openai` でインストール。baseURL パラメータを APIMart エンドポイントに設定し、既存の統合を維持。

```javascript
import OpenAI from 'openai';
const client = new OpenAI({
  baseURL: 'https://api.apimart.ai/v1',
  apiKey: 'your-apimart-key'
});
```

### Java SDK

OpenAI 互換 Java クライアント。簡単な設定変更で全 APIMart モデルとプロバイダーにアクセス可能。

## OpenAI からの移行

5 分以内に APIMart に切り替え可能。コードの書き直し不要——Base URL と API Key の変更のみ。

<Steps>
  <Step title="API Key を取得">
    APIMart に登録し、<a href="https://apimart.ai/console/token" rel="nofollow" target="_blank">コンソールダッシュボード</a>から API Key を取得。
  </Step>
  <Step title="Base URL を更新">
    OpenAI Base URL を APIMart エンドポイントに変更。既存の SDK 統合はそのまま動作。
  </Step>
  <Step title="エンドポイントをテスト">
    新しい Base URL で既存コードを実行。すべての OpenAI 互換エンドポイントは同じ形式で応答。コンソールダッシュボードでレイテンシとレート制限を確認。
  </Step>
</Steps>


## 統一エンドポイント経由でサポートされるモデル

1 つの OpenAI 互換ゲートウェイで主要 LLM プロバイダーにアクセス。すべてのモデルで一貫したリクエスト/レスポンス形式をサポート。

### チャット補完モデル

チャット、補完、コードタスク用の主要言語モデル：

- **GPT-5**：強化された推論機能を持つ OpenAI フラッグシップモデル
- **GPT-4o & GPT-4o Mini**：性能とコストのバランスを取ったマルチモーダルモデル
- **Claude Sonnet 4.5 & Haiku 4.5**：複雑な推論用 Anthropic モデル
- **Gemini 2.0 Flash & Flash Thinking**：Google マルチモーダルモデル

### 画像生成モデル

テキストプロンプトから画像生成：

- **GPT-4o Image**：OpenAI 画像生成モデル
- **Gemini 2.5 Flash Image-preview**：Google 効率的画像モデル

### 動画生成モデル

非同期タスク追跡と Webhook コールバック付き動画生成：

- **OpenAI Sora2**：OpenAI 動画生成モデル
- **Google VEO3**：Google 動画生成モデル

### 音声モデル

OpenAI 互換エンドポイント経由で音声テキスト変換とテキスト音声変換：

- **Whisper-1**：OpenAI 文字起こしモデル
- **TTS**：複数音声のテキスト読み上げ

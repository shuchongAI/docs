## OpenAI 互換エンドポイント

APIMart は OpenAI スタイルのエンドポイントを提供し、シームレスな移行を実現。Base URL を `https://api.apimart.ai/v1` に変更するだけで、既存の SDK をそのまま使用可能。すべてのインターフェースは OpenAI 標準に準拠し、Claude、Gemini、Sora、VEO モデルを拡張サポート。

## 透明な料金体系

サブスクリプション不要の従量課金。公式プロバイダーより手頃な料金で、同じ品質と低レイテンシを維持。

- **透明な料金**：全モデル明確なトークン単価
- **ボリューム割引**：エンタープライズ利用で低料金
- **隠れた費用なし**：表示価格そのまま
- **従量課金**：実際の使用量のみ課金

## OpenAI からの移行

5 分以内に APIMart に切り替え可能。コードの書き直し不要、Base URL と API Key の変更のみ。

<Steps>
  <Step title="API Key を取得">
    APIMart に登録し、<a href="https://api.apimart.ai/console/token" rel="nofollow" target="_blank">コンソールダッシュボード</a>から API Key を取得。
  </Step>
  <Step title="Base URL を更新">
    OpenAI Base URL を `https://api.openai.com/v1` から `https://api.apimart.ai/v1` に変更。既存の SDK 統合はそのまま動作。
  </Step>
  <Step title="エンドポイントをテスト">
    新しい Base URL で既存コードを実行。すべての OpenAI 互換エンドポイントは同じ形式で応答。コンソールダッシュボードでレイテンシとレート制限を確認。
  </Step>
</Steps>


## 統一エンドポイント経由でサポートされるモデル

1 つの OpenAI 互換ゲートウェイで主要 LLM プロバイダーにアクセス。すべてのモデルは `/v1/chat/completions` エンドポイントで利用可能、一貫したリクエスト/レスポンス形式。

### チャット補完モデル

チャット、補完、コードタスク用の主要言語モデル：

- **GPT-5**：強化された推論機能を持つ OpenAI フラッグシップモデル
- **GPT-4o & GPT-4o Mini**：性能とコストのバランスを取ったマルチモーダルモデル
- **Claude Sonnet 4.5 & Haiku 4.5**：複雑な推論用 Anthropic モデル
- **Gemini 2.0 Flash & Flash Thinking**：Google マルチモーダルモデル

### 画像生成モデル

テキストプロンプトから `/v1/images/generations` エンドポイント経由で画像生成：

- **GPT-4o Image**：OpenAI 画像生成モデル
- **Gemini 2.5 Flash Image**：Google 効率的画像モデル

### 動画生成モデル

非同期タスク追跡と Webhook コールバック付き動画生成：

- **OpenAI Sora2**：OpenAI 動画生成モデル
- **Google VEO3**：Google 動画生成モデル

### 音声モデル

OpenAI 互換エンドポイント経由で音声テキスト変換とテキスト音声変換：

- **Whisper-1**：OpenAI 文字起こしモデル
- **TTS**：複数音声のテキスト読み上げ

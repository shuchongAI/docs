# クイックスタート

APIサービスへようこそ！このガイドでは、画像と動画生成を素早く始める方法を説明します。

## ステップ1：APIキーを取得する

1. APIキー管理ページにアクセス：https://your-domain.com/api-keys
2. アカウントにサインインする
3. 新しいAPIキーを作成する
4. キーを安全に保存する（一度だけ表示されます）

## ステップ2：モデルを選択する

複数のAIモデルから選択できます。

### 画像生成モデル
- GPT-4o：高品質な画像生成
- Seedream-4.0：プロフェッショナルな画像生成
- Nano-Banana：高速な画像生成
- Qwen：中国語に最適化された画像生成
- Wan2.5-Image：最新の画像生成技術

### 動画生成モデル
- VEO3：高品質な動画生成
- Sora2：プロフェッショナルな動画生成
- Wan2.5-Video：最新の動画生成技術

## ステップ3：リクエストを送信する

### 画像生成の例

```bash
curl -X POST https://apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-image",
    "prompt": "A cute panda",
    "size": "1024x1024",
    "n": 1
  }'
```

### 動画生成の例

```bash
curl -X POST https://apimart.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3",
    "prompt": "Waves crashing against the shore",
    "duration": 10,
    "resolution": "1920x1080"
  }'
```

## ステップ4：タスクのステータスを確認する

非同期処理を使用しているため、タスクのステータスを照会して結果を取得する必要があります。

```bash
curl -X GET https://apimart.ai/v1/tasks/YOUR_TASK_ID \
  -H "Authorization: Bearer YOUR_API_KEY"
```

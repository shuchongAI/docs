## 截图要求

### 通用要求
- 所有截图应为 PNG 格式
- 保持界面整洁，关闭不必要的弹窗和通知
- 使用高分辨率（至少 1920x1080 for 全屏截图）
- 建议使用浅色主题（如果支持）以保持一致性

### 隐私保护
- 隐藏或模糊真实的 API Key（可使用 `sk-xxxxxxxxxxxxxxxx` 示例）
- 隐藏个人信息、邮箱等敏感数据
- 使用测试数据而非真实业务数据

### 标注建议
- 对于关键操作位置，可以使用红色框或箭头标注
- 保持标注风格一致
- 标注应清晰但不过于突兀

1. 将所有截图放入 `images/integrations/anythingllm/` 目录
2. 确保文件名与文档中引用的名称一致

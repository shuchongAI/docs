# GPT-4o-image Image Generation

Use the GPT-4o-image model for highâ€‘quality image generation. Multiple generation modes are supported.

## Highlights

- Asynchronous processing: returns a task ID for subsequent queries
- Multiple modes: text-to-image, image-to-image, and image editing
- Expiring links: generated image links are valid for 24 hours

## Request example

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-image",
    "prompt": "An ancient castle under the starry sky",
    "size": "1:1",
    "n": 1,
    "image_urls": [
      "https://example.com/image1.png",
      "https://example.com/image2.png"
    ]
  }'
```

</RequestExample>

## Parameters

<ParamField query="model" type="string" required>
  Model name. Use "gpt-4o-image".
</ParamField>

<ParamField query="prompt" type="string" required>
  Text description for image generation.
</ParamField>

<ParamField query="size" type="string">
  Output size. Supported: "1024x1024", "1792x1024", "1024x1792".
</ParamField>

<ParamField query="n" type="integer">
  Number of images to generate. Default: 1.
</ParamField>

<ParamField query="image_urls" type="array">
  Reference image URLs for image-to-image mode.
</ParamField>

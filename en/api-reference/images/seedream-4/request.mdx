# Request Parameters

## Request example

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-4-0",
    "prompt": "An ancient castle under the starry sky",
    "size": "1:1",
    "resolution": "2K",
    "n": 1
  }'
```

</RequestExample>

## Parameters

<ParamField query="model" type="string" required>
  Model name, fixed as `doubao-seedance-4-0`
</ParamField>

<ParamField query="prompt" type="string" required>
  Text description for image generation, maximum 1000 characters
</ParamField>

<ParamField query="size" type="string" default="1:1">
  Image aspect ratio

  Supported ratios: `1:1`, `4:3`, `3:4`, `16:9`, `9:16`, `3:2`, `2:3`, `21:9`, `9:21`
</ParamField>

<ParamField query="resolution" type="string" default="2K">
  Image resolution

  Supported resolutions: `1K`, `2K` (default), `4K`
</ParamField>

<ParamField query="n" type="integer" default="1">
  Number of images to generate, range 1-15
</ParamField>

<ParamField query="image_urls" type="array">
  Reference image URL list for image-to-image mode, maximum 10 images
</ParamField>

<ParamField query="watermark" type="boolean" default="false">
  Whether to add watermark
</ParamField>

<ParamField query="optimize_prompt_options" type="string" default="standard">
  Prompt optimization mode: `standard` or `fast`
</ParamField>

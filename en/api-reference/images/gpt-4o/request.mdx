# Request Parameters

## Request example

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-image",
    "prompt": "An ancient castle under the starry sky",
    "size": "1:1",
    "n": 1,
    "image_urls": [
      "https://cdn.apimart.ai/doc/9998238784524590-549804af-14bc-4fbc-bae5-9d8469d35de3-image_task_01K889V97T2YF8RHQW21XMVMS1_0.png"
    ]
  }'
```

</RequestExample>

## Parameters

<ParamField query="model" type="string" required>
  Model name. Use "gpt-4o-image".
</ParamField>

<ParamField query="prompt" type="string" required>
  Text description for image generation.
</ParamField>

<ParamField query="size" type="string">
  Output size. Supported: "1:1", "1792x1024", "1024x1792".
</ParamField>

<ParamField query="n" type="integer">
  Number of images to generate. Default: 1.
</ParamField>

<ParamField query="image_urls" type="array">
  Reference image URLs for image-to-image mode.
</ParamField>

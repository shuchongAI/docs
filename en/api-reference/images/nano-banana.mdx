# Gemini-2.5-Flash-Image Image Generation

Use the Gemini-2.5-Flash-Image model for fast image generation.

## Highlights

- **Asynchronous processing**: returns a task ID for subsequent queries
- **Fast generation speed**: optimized for rapid image creation
- **Expiring links**: generated image links are valid for 24 hours

## Request example

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-flash-image",
    "prompt": "A bamboo forest path under moonlight",
    "size": "1:1",
    "n": 1
  }'
```

</RequestExample>

## Response example

<ResponseExample>

```json
{
  "code": 200,
  "data": {
    "created": 1757156493,
    "id": "task-unified-1757156512-a7n4jhc2",
    "model": "gemini-2.5-flash-image",
    "object": "image.generation.task",
    "progress": 0,
    "status": "pending",
    "task_info": {
      "can_cancel": true,
      "estimated_time": 60
    },
    "type": "image"
  }
}
```

</ResponseExample>

## Parameters

<ParamField query="model" type="string" required>
  Model name, fixed as "gemini-2.5-flash-image"
</ParamField>

<ParamField query="prompt" type="string" required>
  Text description for image generation
</ParamField>

<ParamField query="size" type="string">
  Image dimensions. Supported: "1024x1024", "1024x1536", "1536x1024"
</ParamField>

<ParamField query="n" type="integer">
  Number of images to generate. Default: 1
</ParamField>

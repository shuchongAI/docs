# VEO3 Video Generation

Use the VEO3 model for highâ€‘quality video generation.

## Highlights

- Asynchronous processing: returns a task ID for subsequent queries
- Multiple modes: text-to-video, image-to-video
- Expiring links: generated video links are valid for 24 hours

## Request example

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1",
    "prompt": "A group of dolphins leaping in the azure ocean",
    "duration": 10,
    "aspect_ratio": "16:9"
  }'
```

</RequestExample>

## Parameters

<ParamField query="model" type="string" required>
  Model name. Use "veo3".
</ParamField>

<ParamField query="prompt" type="string" required>
  Text description for video generation.
</ParamField>

<ParamField query="duration" type="integer">
  Video duration (seconds). Default: 10.
</ParamField>

<ParamField query="aspect_ratio" type="string">
  Video aspect_ratio

  Supported formats:
  - `16:9` (Landscape)
  - `9:16` (Portrait)
</ParamField>

<ParamField query="generation_type" type="string">
  Video generation type

  Supported types:
  - `frame` - Frame-to-video (FL mode)
  - `reference` - Reference image-to-video
</ParamField>

<ParamField query="image_urls" type="array">
  URL list of reference images for image-to-video generation

  - Maximum 3 images
  - For frame-to-video mode: first image is the start frame, second image is the end frame
  - Supported formats: .jpeg, .jpg, .png, .webp
</ParamField>

# Wan2.5-Video Generation

Use the Wan2.5-Video model for high‑quality video generation.

## Highlights

- Asynchronous processing: returns a task ID for subsequent queries
- Multiple modes: text-to-video, image-to-video
- Expiring links: generated video links are valid for 24 hours

## Request example

<RequestExample>

```bash
curl -X POST https://apimart.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.5-video",
    "prompt": "City traffic at night",
    "duration": 12,
    "resolution": "1920x1080"
  }'
```

</RequestExample>

## Parameters

<ParamField query="model" type="string" required>
  Model name. Use "wan2.5-video".
</ParamField>

<ParamField query="prompt" type="string" required>
  Text description for video generation.
</ParamField>

<ParamField query="duration" type="integer">
  Video duration (seconds). Default: 12.
</ParamField>

<ParamField query="resolution" type="string">
  Video resolution. Supported: "1920x1080", "1280x720".
</ParamField>

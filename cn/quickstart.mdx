# 快速开始

欢迎使用我们的API服务！本指南将帮助您快速开始使用图像和视频生成功能。

## 第一步：获取API密钥

1. 访问 [API密钥管理页面](https://your-domain.com/api-keys)
2. 登录您的账户
3. 创建新的API密钥
4. 保存您的密钥（密钥只显示一次）

## 第二步：选择模型

我们提供多种AI模型供您选择：

### 图像生成模型
- **GPT-4o**: 高质量图像生成
- **Seedream-4.0**: 专业图像生成
- **Nano-Banana**: 快速图像生成
- **Qwen**: 中文优化图像生成
- **Wan2.5-Image**: 最新图像生成技术

### 视频生成模型
- **VEO3**: 高质量视频生成
- **Sora2**: 专业视频生成
- **Wan2.5-Video**: 最新视频生成技术

## 第三步：发送请求

### 图像生成示例

```bash
curl -X POST https://api.evolink.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-image",
    "prompt": "一只可爱的熊猫",
    "size": "1024x1024",
    "n": 1
  }'
```

### 视频生成示例

```bash
curl -X POST https://api.evolink.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3",
    "prompt": "海浪拍打着海岸",
    "duration": 10,
    "resolution": "1920x1080"
  }'
```

## 第四步：查询任务状态

由于我们使用异步处理模式，您需要查询任务状态来获取结果：

```bash
curl -X GET https://api.evolink.ai/v1/tasks/YOUR_TASK_ID \
  -H "Authorization: Bearer YOUR_API_KEY"
```

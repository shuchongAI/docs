## Q7: 支持哪些 AI 模型？

APIMart 支持多种主流 AI 模型，包括：

- **文本模型**：GPT-5, Claude, Gemini 等
- **图像模型**：GPT-4o-image, Gemini-2.5-Flash-Image 等
- **视频模型**：Sora2, VEO3 等
- **音频模型**：Whisper-1, TTS 等

具体支持的模型请查看 [API 文档](/cn/index)。

## Q8: 与 OpenAI API 的兼容性如何？

APIMart 完全兼容 OpenAI API 格式，您可以：

- 直接替换 API 端点
- 使用相同的请求格式
- 无需修改现有代码
- 支持所有主要的 OpenAI API 功能

<Info>
只需将原有的 OpenAI API 端点 `https://api.openai.com` 替换为 `https://api.apimart.ai`，即可无缝迁移。
</Info>

## 如何优化 API 调用成本？

优化成本的建议：

- 选择适合您需求的模型（不同模型价格不同）
- 使用流式响应减少等待时间
- 合理设置 token 限制
- 缓存常见的响应结果
- 监控使用情况避免意外消耗

<Tip>
**实用技巧**

1. **选择合适的模型**：对于简单任务，使用成本较低的模型即可获得良好效果
2. **设置 max_tokens**：限制响应长度可以有效控制成本
3. **实施缓存策略**：对于重复或相似的请求，缓存结果可以大幅降低调用次数
4. **使用流式响应**：提升用户体验的同时，可以更早发现问题并中断不必要的请求
</Tip>

## 无法连接到 APIMart 服务？

如果您无法连接到 APIMart 服务，请检查以下几点：

- 确认您的 API Key 是否正确配置
- 检查网络连接是否正常
- 确认 API 端点地址是否正确：`https://api.apimart.ai`
- 检查防火墙或代理设置是否阻止了连接

## 模型选择后没有响应或报错？

这可能是由于以下原因：

- API Key 没有正确配置或已过期
- 选择的模型可能暂时不可用
- 请求格式不正确
- 账户余额不足

请前往 [APIMart 控制台](https://apimart.ai) 检查您的账户状态和 API 密钥。

## 对话出现错误或中断怎么办？

遇到对话错误时：

- 检查错误消息中的具体信息
- 确认请求参数是否符合 API 规范
- 查看 API 文档了解正确的请求格式
- 如果问题持续，请联系技术支持

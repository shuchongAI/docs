## 从 OpenAI 迁移

5 分钟内切换到 APIMart。无需重写代码，只需更改 Base URL 和 API Key。

<Steps>
  <Step title="获取 API Key">
    注册 APIMart 并从<a href="https://api.apimart.ai/console/token" rel="nofollow" target="_blank">控制台</a>获取 API Key。
  </Step>
  <Step title="更新 Base URL">
    将 OpenAI Base URL 从 `https://api.openai.com/v1` 改为 `https://api.apimart.ai/v1`。现有 SDK 集成无需修改即可工作。
  </Step>
  <Step title="测试端点">
    使用新的 Base URL 运行现有代码。所有 OpenAI 兼容端点响应格式完全相同。在控制台查看延迟和速率限制。
  </Step>
</Steps>


## 统一端点支持的模型

通过一个 OpenAI 兼容网关访问领先的 LLM 供应商。所有模型可通过 `/v1/chat/completions` 端点访问，请求/响应格式一致。

### 聊天补全模型

用于聊天、补全和代码任务的领先语言模型：

- **GPT-5**：OpenAI 旗舰模型，增强推理能力
- **GPT-4o & GPT-4o Mini**：多模态模型，平衡性能与成本
- **Claude Sonnet 4.5 & Haiku 4.5**：Anthropic 复杂推理模型
- **Gemini 2.0 Flash & Flash Thinking**：Google 多模态模型

### 图像生成模型

通过 `/v1/images/generations` 端点从文本提示生成图像：

- **GPT-4o Image**：OpenAI 图像生成模型
- **Gemini 2.5 Flash Image**：Google 高效图像模型

### 视频生成模型

支持异步任务追踪和 Webhook 回调的视频生成：

- **OpenAI Sora2**：OpenAI 视频生成模型
- **Google VEO3**：Google 视频生成模型

### 语音模型

通过 OpenAI 兼容端点实现语音转文本和文本转语音：

- **Whisper-1**：OpenAI 转录模型
- **TTS**：多种音色的文本转语音

# 请求参数

## 请求示例

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-flash-image-preview",
    "prompt": "月光下的竹林小径",
    "size": "1:1",
    "n": 1
  }'
```

</RequestExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 "gemini-2.5-flash-image-preview"
</ParamField>

<ParamField query="prompt" type="string" required>
  图像生成的文本描述
</ParamField>

<ParamField query="size" type="string">
  生成图像的尺寸，支持 "1:1", "1024x1536", "1536x1024"
</ParamField>

<ParamField query="n" type="integer">
  生成图像的数量，默认为 1
</ParamField>

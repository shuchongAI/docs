# GPT-4o-image 图像生成

使用 GPT-4o-image 模型进行高质量的图像生成，支持多种生成模式。

## 功能特点

- **异步处理模式**：返回任务ID用于后续查询
- **支持多种生成模式**：文本转图片、图生图、图像编辑等
- **生成的图像链接**：有效期为24小时，请尽快保存

## 请求示例

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-image",
    "prompt": "星空下的古老城堡",
    "size": "1:1",
    "n": 1,
    "image_urls": [
      "https://example.com/image1.png",
      "https://example.com/image2.png"
    ]
  }'
```

</RequestExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 "gpt-4o-image"
</ParamField>

<ParamField query="prompt" type="string" required>
  图像生成的文本描述
</ParamField>

<ParamField query="size" type="string">
  生成图像的尺寸，支持 "1:1", "1792x1024", "1024x1792"
</ParamField>

<ParamField query="n" type="integer">
  生成图像的数量，默认为 1
</ParamField>

<ParamField query="image_urls" type="array">
  参考图像URL列表，用于图生图模式
</ParamField>

# 请求参数

## 请求示例

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-4-0",
    "prompt": "星空下的古老城堡",
    "size": "1:1",
    "resolution": "2K",
    "n": 1
  }'
```

</RequestExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 `doubao-seedance-4-0`
</ParamField>

<ParamField query="prompt" type="string" required>
  图像生成的文本描述，最长 1000 个字符
</ParamField>

<ParamField query="size" type="string" default="1:1">
  图像宽高比

  支持的比例：`1:1`、`4:3`、`3:4`、`16:9`、`9:16`、`3:2`、`2:3`、`21:9`、`9:21`
</ParamField>

<ParamField query="resolution" type="string" default="2K">
  图像分辨率

  支持的分辨率：`1K`、`2K`（默认）、`4K`
</ParamField>

<ParamField query="n" type="integer" default="1">
  生成图像的数量，范围 1-15
</ParamField>

<ParamField query="image_urls" type="array">
  参考图像 URL 列表，用于图生图模式，最多 10 张
</ParamField>

<ParamField query="watermark" type="boolean" default="false">
  是否添加水印
</ParamField>

<ParamField query="optimize_prompt_options" type="string" default="standard">
  提示词优化模式：`standard`（标准）或 `fast`（快速）
</ParamField>

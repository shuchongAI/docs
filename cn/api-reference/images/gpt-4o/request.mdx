# 请求参数

## 请求示例

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o-image",
    "prompt": "星空下的古老城堡",
    "size": "1:1",
    "n": 1,
    "image_urls": [
      "https://cdn.apimart.ai/doc/9998238784524590-549804af-14bc-4fbc-bae5-9d8469d35de3-image_task_01K889V97T2YF8RHQW21XMVMS1_0.png"
    ]
  }'
```

</RequestExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 "gpt-4o-image"
</ParamField>

<ParamField query="prompt" type="string" required>
  图像生成的文本描述
</ParamField>

<ParamField query="size" type="string">
  生成图像的尺寸，支持 "1024x1024", "1792x1024", "1024x1792"
</ParamField>

<ParamField query="n" type="integer">
  生成图像的数量，默认为 1
</ParamField>

<ParamField query="image_urls" type="array">
  参考图像URL列表，用于图生图模式
</ParamField>

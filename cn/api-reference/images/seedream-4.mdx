# Seedream-4.0 图像生成

使用 Seedream-4.0 模型进行高质量的图像生成。

## 功能特点

- **异步处理模式**：返回任务ID用于后续查询
- **支持多种生成模式**：文本转图片、图生图、图像编辑等
- **生成的图像链接**：有效期为24小时，请尽快保存

## 请求示例

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedream-4.0",
    "prompt": "一只可爱的熊猫在竹林中玩耍",
    "size": "1:1",
    "n": 1
  }'
```

</RequestExample>

## 响应示例

<ResponseExample>

{
  "code": 200,

    "id": "task-unified-1757156567-f5xt9bk6",

</ResponseExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 "seedream-4.0"
</ParamField>

<ParamField query="prompt" type="string" required>
  图像生成的文本描述
</ParamField>

<ParamField query="size" type="string">
  生成图像的尺寸，支持 "1024x1024", "1792x1024", "1024x1792"
</ParamField>

<ParamField query="n" type="integer">
  生成图像的数量，默认为 1
</ParamField>

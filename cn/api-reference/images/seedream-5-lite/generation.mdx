## Body

<ParamField body="model" type="string" default="doubao-seedream-5-0-lite" required>
  图像生成模型名称

  Example: `"doubao-seedream-5-0-lite"`
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文本描述
</ParamField>

<ParamField body="size" type="string" default="1:1">
  图像生成的宽高比

  支持的宽高比：
  - `1:1` - 正方形（默认）
  - `4:3` - 横向 4:3
  - `3:4` - 纵向 3:4
  - `16:9` - 横向宽屏
  - `9:16` - 纵向竖屏
  - `3:2` - 横向 3:2
  - `2:3` - 纵向 2:3
  - `21:9` - 超宽屏
  - `auto` - 自动匹配参考图像比例（需要提供 image_urls）

  > **注意：** Seedream-5.0-Lite 不支持 `9:21` 比例
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  图像分辨率

  支持的分辨率：
  - `2K` - 标准分辨率（默认）
  - `3K` - 高清分辨率

  > **注意：** Seedream-5.0-Lite 仅支持 2K 和 3K 分辨率，不支持 1K 或 4K

  **分辨率参考尺寸：**
  | 分辨率 | 1:1 尺寸 | 16:9 尺寸 |
  |--------|----------|-----------|
  | 2K | 2048x2048 | 2848x1600 |
  | 3K | 3072x3072 | 4096x2304 |

  <Expandable title="完整像素映射表">
    **2K 分辨率**
    | 比例 | 像素 |
    |------|------|
    | 1:1 | 2048×2048 |
    | 4:3 | 2304×1728 |
    | 3:4 | 1728×2304 |
    | 16:9 | 2848×1600 |
    | 9:16 | 1600×2848 |
    | 3:2 | 2496×1664 |
    | 2:3 | 1664×2496 |
    | 21:9 | 3136×1344 |

    **3K 分辨率**
    | 比例 | 像素 |
    |------|------|
    | 1:1 | 3072×3072 |
    | 4:3 | 3456×2592 |
    | 3:4 | 2592×3456 |
    | 16:9 | 4096×2304 |
    | 9:16 | 2304×4096 |
    | 3:2 | 3744×2496 |
    | 2:3 | 2496×3744 |
    | 21:9 | 4704×2016 |
  </Expandable>
</ParamField>

<ParamField body="n" type="integer">
  生成图像的数量

  取值范围：1-4

  默认值：1

  **注意：**
  - 当 `n > 1` 时自动启用组图模式
  - **必须输入纯数字（如 `1`），不要加引号，否则会报错**

  会根据生成数量进行预扣费
</ParamField>

<ParamField body="image_urls" type="array">
  参考图像的 URL 列表，用于图生图

  支持两种格式：

  **1. 完整的图像URL地址**
  - 公开可访问的图像URL（http:// 或 https://）
  - 示例：`https://example.com/image.jpg`

  **2. Base64 编码格式**
  - **必须使用完整的 Data URI 格式**
  - 格式：`data:image/{格式};base64,{base64数据}`
  - 支持的图片格式：jpeg、png
  - 示例：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`
  - ⚠️ 注意：必须包含 `data:image/jpeg;base64,` 前缀部分

  **限制：**
  - 图片格式：jpeg、png
  - 宽高比（宽/高）范围：[1/3, 3]
  - 宽高长度（px）> 14
  - 单张图片大小：不超过 10MB
  - 总像素：不超过 6000×6000 px
</ParamField>

<ParamField body="output_format" type="string" default="jpeg">
  输出图片格式

  - `jpeg`：JPEG 格式（默认）
  - `png`：PNG 格式，适用于需要透明背景等场景

  > **注意：** `output_format` 参数为 Seedream-5.0-Lite 特有功能，其他图片模型传入会被忽略
</ParamField>

<ParamField body="sequential_image_generation" type="string" default="disabled">
  组图生成模式

  用于控制是否生成多张图片：

  - `disabled`：禁用组图模式（默认）
  - `auto`：启用组图模式，可以生成多张图片

  **使用说明：**
  - 当 `n > 1` 时会自动设置为 `auto`
</ParamField>

<ParamField body="sequential_image_generation_options" type="object">
  组图生成选项

  当 `sequential_image_generation` 设置为 `auto` 时可用

  **属性：**
  - `max_images` (integer): 指定生成图片的数量

  **示例：**
  ```json
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 4
  }
  ```
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  是否在生成的图像上添加水印

  - `true`：添加水印
  - `false`：不添加水印（默认）
</ParamField>

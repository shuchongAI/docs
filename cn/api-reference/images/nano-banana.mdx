# Nano-Banana 图像生成

使用 Nano-Banana 模型进行快速图像生成。

## 功能特点

- **异步处理模式**：返回任务ID用于后续查询
- **快速生成**：相比其他模型生成速度更快
- **生成的图像链接**：有效期为24小时，请尽快保存

## 请求示例

<RequestExample>

```bash
curl -X POST https://apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana",
    "prompt": "现代城市夜景，霓虹灯闪烁",
    "size": "1024x1024",
    "n": 1
  }'
```

</RequestExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 "nano-banana"
</ParamField>

<ParamField query="prompt" type="string" required>
  图像生成的文本描述
</ParamField>

<ParamField query="size" type="string">
  生成图像的尺寸，支持 "1024x1024", "1792x1024", "1024x1792"
</ParamField>

<ParamField query="n" type="integer">
  生成图像的数量，默认为 1
</ParamField>

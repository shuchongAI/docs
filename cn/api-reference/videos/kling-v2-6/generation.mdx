## 请求参数

<ParamField body="model" type="string" required>
  视频生成模型名称

  支持的模型：
  - `kling-v2-6` - 可灵 v2.6（推荐）
</ParamField>

<ParamField body="prompt" type="string" required>
  视频内容描述

  建议详细描述场景、动作、风格等，以获得更好的生成效果

  示例：`"一只金色的猫咪在阳光下的草地上奔跑，慢动作，电影质感"`
</ParamField>

<ParamField body="mode" type="string" default="std">
  生成模式

  可选值：
  - `std` - 标准模式（720P，仅支持无声视频）
  - `pro` - 专业模式（1080P，支持音频生成和音色控制）

  默认值：`std`

  <Warning>
  **标准模式限制**：`std` 模式仅支持无声视频，`audio`、`voice_list` 参数需在 `pro` 模式下使用。
  </Warning>
</ParamField>

<ParamField body="duration" type="integer" default="5">
  视频时长（秒）

  可选值：`5` 或 `10`

  默认值：`5`
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  视频宽高比

  可选值：
  - `16:9` - 横屏
  - `9:16` - 竖屏
  - `1:1` - 方形

  默认值：`16:9`
</ParamField>

<ParamField body="negative_prompt" type="string">
  负面提示词，用于排除不想要的内容

  示例：`"模糊, 低画质, 变形"`
</ParamField>

<ParamField body="image_urls" type="array<url>">
  图片 URL 数组，用于图生视频

  - 传 **1 张**：作为首帧图片
  - 传 **2 张**：自动分配为首帧 + 尾帧（需 `mode: "pro"`）

  最多支持 2 张图片

  示例：`["https://example.com/first.jpg"]`

  <Warning>
  - 最多支持 2 张图片
  - 首尾帧功能（传 2 张图片）仅支持 `pro` 模式，`std` 模式仅支持首帧（1 张图片）
  - 图生视频时，`aspect_ratio` 可能被图片实际比例覆盖
  </Warning>
</ParamField>

<ParamField body="audio" type="boolean" default="false">
  是否自动生成音频

  默认值：`false`

  <Warning>
  仅在 `mode: "pro"` 下可用
  </Warning>
</ParamField>

<ParamField body="voice_list" type="array">
  指定音色列表，最多 2 个音色

  <Expandable title="字段说明">
    <ParamField body="voice_id" type="string" required>
      预设音色 ID，可通过音色查询接口获取
    </ParamField>
  </Expandable>

  示例：
  ```json
  [{"voice_id": "829824295735410756"}]
  ```

  <Warning>
  需要同时满足 `mode: "pro"` 和 `audio: true` 才能使用
  </Warning>
</ParamField>

<ParamField body="watermark" type="boolean">
  是否添加水印
</ParamField>

## 功能支持矩阵

| 类型 | 功能 | std 5s | std 10s | pro 5s | pro 10s |
|------|------|--------|---------|--------|---------|
| 文生视频 | 视频生成 | ✅（仅无声） | ✅（仅无声） | ✅ | ✅ |
| 图生视频 | 视频生成 | ✅（仅无声） | ✅（仅无声） | ✅ | ✅ |
| 图生视频 | 首帧 | ✅ | ✅ | ✅ | ✅ |
| 图生视频 | 尾帧 | - | - | ✅ | ✅ |
| 图生视频 | 声音控制 | - | - | ✅ | ✅ |

## 文生视频 vs 图生视频

系统通过是否传入 `image_urls` 自动判断模式：不传图片为文生视频，传图片为图生视频。

| 参数 | 文生视频 | 图生视频 |
|------|---------|---------|
| `prompt` | ✅ 必填 | ✅ 必填 |
| `image_urls` | ❌ 不传 | ✅ 必填（1-2 张，尾帧需 `pro`） |
| `negative_prompt` | ✅ 可选 | ✅ 可选 |
| `mode` | ✅ 可选 | ✅ 可选 |
| `duration` | ✅ 可选 | ✅ 可选 |
| `aspect_ratio` | ✅ 可选 | ⚠️ 可能被图片比例覆盖 |
| `audio` / `voice_list` | ✅ 可选（需 `pro`） | ✅ 可选（需 `pro`） |
| `watermark` | ✅ 可选 | ✅ 可选 |

## 使用场景

### 场景 1：文生视频（标准模式）

```json
{
  "model": "kling-v2-6",
  "prompt": "一只金色的猫咪在阳光下的草地上奔跑，慢动作，电影质感",
  "mode": "std",
  "duration": 5,
  "aspect_ratio": "16:9"
}
```

### 场景 2：文生视频（专业模式 + 负面提示词）

```json
{
  "model": "kling-v2-6",
  "prompt": "东京涩谷十字路口，雨夜霓虹灯倒映在湿漉漉的地面上，行人撑伞穿行",
  "negative_prompt": "模糊, 低画质, 变形",
  "mode": "pro",
  "duration": 10,
  "aspect_ratio": "16:9"
}
```

### 场景 3：图生视频（首帧图片）

```json
{
  "model": "kling-v2-6",
  "prompt": "让画面中的人物转头微笑",
  "image_urls": ["https://example.com/portrait.jpg"],
  "mode": "std",
  "duration": 5,
  "aspect_ratio": "16:9"
}
```

### 场景 4：图生视频（首帧 + 尾帧控制）

```json
{
  "model": "kling-v2-6",
  "prompt": "从白天过渡到黑夜的城市延时摄影",
  "image_urls": ["https://example.com/day-city.jpg", "https://example.com/night-city.jpg"],
  "mode": "pro",
  "duration": 5
}
```

### 场景 5：专业模式 + 自动音频

```json
{
  "model": "kling-v2-6",
  "prompt": "海浪拍打礁石，海鸥在空中盘旋，远处有灯塔",
  "mode": "pro",
  "duration": 10,
  "audio": true,
  "aspect_ratio": "16:9"
}
```

### 场景 6：专业模式 + 指定音色

```json
{
  "model": "kling-v2-6",
  "prompt": "一位主持人面对镜头播报新闻",
  "image_urls": ["https://example.com/anchor.jpg"],
  "mode": "pro",
  "duration": 10,
  "audio": true,
  "voice_list": [{"voice_id": "829826792415842333"}],
  "aspect_ratio": "16:9"
}
```

<Note>
**查询任务结果**

视频生成为异步任务，提交后会返回 `task_id`。使用 [获取任务状态](/cn/api-reference/tasks/status) 接口查询生成进度和结果。
</Note>

# Wan2.5-Video 视频生成

使用 Wan2.5-Video 模型进行高质量的视频生成。

## 功能特点

- **异步处理模式**：返回任务ID用于后续查询
- **支持多种生成模式**：文本转视频、图生视频等
- **生成的视频链接**：有效期为24小时，请尽快保存

## 请求示例

<RequestExample>

```bash
curl -X POST https://apimart.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.5-video",
    "prompt": "城市夜景中的车流",
    "duration": 12,
    "resolution": "1920x1080"
  }'
```

</RequestExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 "wan2.5-video"
</ParamField>

<ParamField query="prompt" type="string" required>
  视频生成的文本描述
</ParamField>

<ParamField query="duration" type="integer">
  视频时长（秒），默认为 12
</ParamField>

<ParamField query="resolution" type="string">
  视频分辨率，支持 "1920x1080", "1280x720"
</ParamField>

## 请求参数

<ParamField body="model" type="string" required>
  视频生成模型名称

  固定值：`MiniMax-Hailuo-02`
</ParamField>

<ParamField body="prompt" type="string" required>
  视频内容描述

  建议详细描述场景、动作、风格等，以获得更好的生成效果

  示例：`"一只可爱的猫咪在草地上奔跑"`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  视频时长（秒）

  可选值：
  - `5` - 5秒视频
  - `10` - 10秒视频

  默认值：`5`

  <Warning>
  **1080p 限制**：使用 1080p 分辨率时，仅支持 5 秒时长
  </Warning>
</ParamField>

<ParamField body="resolution" type="string" default="768p">
  视频分辨率

  可选值：
  - `512p` - 标清
  - `768p` - 高清
  - `1080p` - 全高清（仅支持 6 秒时长）

  默认值：`768p`
</ParamField>

<ParamField body="prompt_optimizer" type="boolean" default="true">
  是否自动优化 prompt

  启用后，系统会自动优化您的提示词以获得更好的生成效果

  默认值：`true`
</ParamField>

<ParamField body="fast_pretreatment" type="boolean" default="false">
  是否缩短 prompt 优化耗时

  启用后可加快处理速度，但可能略微影响优化效果

  默认值：`false`
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  是否添加水印

  默认值：`false`
</ParamField>

<ParamField body="first_frame_image" type="string">
  视频起始帧图片

  支持两种格式：
  - **公网 URL**：`https://example.com/start.jpg`
  - **Base64 编码**：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`

  用于指定视频的起始画面
</ParamField>

<ParamField body="last_frame_image" type="string">
  视频结束帧图片

  支持两种格式：
  - **公网 URL**：`https://example.com/end.jpg`
  - **Base64 编码**：`data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg...`

  用于指定视频的结束画面
</ParamField>

## 参数限制

| 限制项 | 说明 |
|--------|------|
| 时长 | 仅支持 5 或 10 秒 |
| 1080p 分辨率 | 仅支持 5 秒时长 |
| 图片格式 | 支持公网 URL 或 Base64 编码（`data:image/jpeg;base64,...`） |

## 分辨率与时长组合

| 分辨率 | 支持的时长 | 备注 |
|--------|-----------|------|
| 512p | 5秒、10秒 | 全部支持 |
| 768p | 5秒、10秒 | 全部支持 |
| 1080p | 5秒 | 不支持10秒 |

## 使用场景

### 场景 1：快速生成文本视频

```json
{
  "model": "MiniMax-Hailuo-02",
  "prompt": "一只可爱的猫咪在草地上奔跑，阳光明媚"
}
```

### 场景 2：生成高质量 1080p 视频

```json
{
  "model": "MiniMax-Hailuo-02",
  "prompt": "城市夜景，霓虹灯闪烁，车流穿梭",
  "duration": 5,
  "resolution": "1080p",
  "prompt_optimizer": true,
  "watermark": false
}
```

### 场景 3：基于首帧图生成视频

```json

  "duration": 5,
  "resolution": "768p",
  "first_frame_image": "https://example.com/portrait.jpg"
```

<Note>
**查询任务结果**

视频生成为异步任务，提交后会返回 `task_id`。使用 [获取任务状态](/cn/api-reference/tasks/status) 接口查询生成进度和结果。
</Note>

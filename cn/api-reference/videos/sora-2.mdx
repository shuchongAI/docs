# Sora-2 视频生成

使用 Sora-2 模型进行高质量的视频生成。

## 功能特点

- **异步处理模式**：返回任务ID用于后续查询
- **支持多种生成模式**：文本转视频、图生视频等
- **生成的视频链接**：有效期为24小时，请尽快保存

## 请求示例

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "sora-2",
    "prompt": "瀑布倾泻而下，彩虹横跨山谷",
    "duration": 15,
    "resolution": "1920x1080"
  }'
```

</RequestExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 "sora-2"
</ParamField>

<ParamField query="prompt" type="string" required>
  视频生成的文本描述
</ParamField>

<ParamField query="duration" type="integer">
  视频时长（秒），支持 10 或 15
</ParamField>

<ParamField query="resolution" type="string">
  视频分辨率，支持 "1920x1080", "1280x720"
</ParamField>

# Wan2.5 首顺图生视频

使用 Wan2.5-Video 模型从静态图像生成动态视频。

## 功能特点

- **图像驱动**：基于输入图像生成连贯的视频内容
- **智能补间**：自动生成流畅的过渡帧
- **保持风格**：保持原始图像的视觉风格

## 请求示例

<RequestExample>

```bash
curl -X POST https://api.evolink.ai/v1/videos/image-to-video \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.5-video",
    "image": "https://example.com/image.png",
    "prompt": "让画面中的人物微笑并挥手",
    "duration": 5,
    "resolution": "1920x1080"
  }'
```

</RequestExample>

## 参数说明

<ParamField query="model" type="string" required>
  模型名称，固定为 "wan2.5-video"
</ParamField>

<ParamField query="image" type="string" required>
  输入图像URL
</ParamField>

<ParamField query="prompt" type="string" required>
  视频生成的动作描述
</ParamField>

<ParamField query="duration" type="integer">
  视频时长（秒），默认为 5
</ParamField>

<ParamField query="resolution" type="string">
  视频分辨率
</ParamField>

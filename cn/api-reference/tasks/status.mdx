# 查询任务状态

查询异步任务的执行状态和结果。

## 功能特点

- **实时状态查询**：获取任务的最新执行状态
- **进度跟踪**：显示任务完成进度
- **结果获取**：任务完成后获取生成结果

## 请求示例

<RequestExample>

```bash
curl -X GET https://api.evolink.ai/v1/tasks/task-unified-1757156493-imcg5zqt \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</RequestExample>

## 响应示例

<ResponseExample>

```json
{
  "id": "task-unified-1757156493-imcg5zqt",
  "status": "completed",
  "progress": 100,
  "result": {
    "images": [
      {
        "url": "https://example.com/generated-image.png",
        "expires_at": 1757242893
      }
    ]
  },
  "created": 1757156493,
  "completed": 1757156593,
  "estimated_time": 100,
  "actual_time": 100
}
```

</ResponseExample>

## 参数说明

<ParamField path="task_id" type="string" required>
  任务ID，由生成接口返回
</ParamField>

## 状态说明

<ResponseField name="status" type="string">
  任务状态：
  - `pending`: 等待中
  - `processing`: 处理中
  - `completed`: 已完成
  - `failed`: 失败
  - `cancelled`: 已取消
</ResponseField>

<ResponseField name="progress" type="integer">
  任务进度百分比 (0-100)
</ResponseField>

<ResponseField name="result" type="object">
  任务结果，仅在状态为 `completed` 时返回
</ResponseField>

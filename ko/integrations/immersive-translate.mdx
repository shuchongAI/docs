## 사전 준비

시작하기 전에 다음을 확인하세요:

1. **Immersive Translate 확장 프로그램 설치 완료**  
   다음에서 설치:
   - [Chrome 웹 스토어](https://chrome.google.com/webstore/detail/immersive-translate/bpoadfkcbjbfhfodiogcnhhhpibjhbnh)
   - [Edge 애드온](https://microsoftedge.microsoft.com/addons/detail/immersive-translate/amkbmndfnliijdhojkpoglbnaaahippg)
   - [Firefox 애드온](https://addons.mozilla.org/firefox/addon/immersive-translate/)
   - [공식 웹사이트](https://immersivetranslate.com/)

2. **APIMart API 키 획득 완료**  
   [APIMart 콘솔](https://apimart.ai/console/token)에 로그인하여 API 키(`sk-`로 시작) 획득

<Note>
**팁:** APIMart 계정이 없는 경우, 먼저 [APIMart](https://apimart.ai)에서 등록하고 API 키를 받으세요.
</Note>

## 1단계: 확장 프로그램 설정 열기

Immersive Translate 확장 프로그램 설치 후:

1. 브라우저 도구 모음에서 **Immersive Translate** 아이콘 클릭
2. 팝업 패널 하단의 **설정** 버튼 클릭
3. Immersive Translate 설정 페이지로 이동

## 2단계: 번역 서비스 구성

### 2.1 번역 서비스 설정 진입

설정 페이지에서:

1. 왼쪽 메뉴에서 **"번역 서비스"** 옵션 찾기
2. 클릭하여 번역 서비스 구성 페이지로 이동

### 2.2 사용자 정의 번역 서비스 추가

번역 서비스 목록 페이지에서:

1. 상단의 **+ 사용자 정의 번역 서비스 추가** 버튼 클릭
2. 구성 양식에 APIMart API 정보 입력

### 2.3 APIMart API 정보 구성

다음 구성 정보를 입력하세요:

| 구성 항목 | 값 | 설명 |
|----------|-----|------|
| **사용자 정의 번역 서비스 이름** | `APIMart` | 식별하기 쉽게 사용자 정의 가능 |
| **APIKEY** | `sk-xxxxxxxxxxxx` | 귀하의 APIMart API 키 |
| **모델** | `gpt-4o-mini` 등 | 드롭다운 목록에서 선택 |
| **사용자 정의 API 엔드포인트** | `https://api.apimart.ai/v1/chat/completions` | 전체 API 엔드포인트 주소 |
| **초당 최대 요청 수** | `5` | 요청 빈도 제어, 3-5 권장 |

<Note>
**구성 참고:**
- **APIKEY**: [APIMart 콘솔](https://apimart.ai/console/token)에서 획득하고 `sk-`로 시작해야 합니다
- **API 엔드포인트**: 전체 경로 `https://api.apimart.ai/v1/chat/completions` 사용 필수
- **모델 선택**: `gpt-4o-mini`(가성비 좋음) 또는 `gpt-5`(최고 품질) 권장
</Note>

## 3단계: 저장 및 번역 서비스 선택

구성 완료 후:

1. 구성 페이지 하단의 **저장** 버튼 클릭
2. 확장 프로그램 패널로 돌아가서 **번역 서비스** 드롭다운 메뉴에서 **APIMart** 선택
3. **AI 전문가**에서 번역 스타일(예: "일반") 선택 가능

### 권장 모델

다양한 모델이 다양한 번역 시나리오에 적합합니다:

- **gpt-5** - 최신 모델, 최고 번역 품질 ⭐ 권장
- **gpt-4o** - 고품질 번역, 빠른 속도
- **gpt-4o-mini** - 가성비 좋음, 대량 번역에 적합 💰 최고 가치
- **claude-sonnet-4-5-20250929** - 문학 번역에 적합
- **claude-haiku-4-5-20251001** - 빠른 번역, 가성비 좋음

## 기능

Immersive Translate + APIMart를 사용하면:

- 🌐 **웹페이지 번역** - 이중 언어 비교 표시, 원본 레이아웃 유지
- 📄 **PDF 번역** - 온라인 PDF 문서 번역 지원
- 🎬 **비디오 자막** - YouTube, Netflix 등의 이중 언어 자막
- ✍️ **단어 선택 번역** - 선택하고 번역, 빠른 이해
- 📱 **전 플랫폼 지원** - Chrome, Edge, Firefox, Safari 등

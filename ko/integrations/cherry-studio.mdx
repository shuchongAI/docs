## 사전 준비

시작하기 전에 다음을 확인하세요:

1. **Cherry Studio 설치 완료**  
   [Cherry Studio 공식 웹사이트](https://cherry-ai.com/)에서 운영 체제에 적합한 버전을 다운로드하고 설치하세요

2. **APIMart API 키 획득**  
   [APIMart 콘솔](https://apimart.ai/console/token)에 로그인하여 API 키(`sk-`로 시작)를 받으세요

<Note>
**팁:** APIMart 계정이 없는 경우 먼저 [APIMart](https://apimart.ai)에서 등록하고 API 키를 받으세요.
</Note>

## 1단계: Cherry Studio 설정 열기

Cherry Studio를 실행한 후 설정 페이지로 이동합니다:

1. 오른쪽 상단의 **⚙️ 설정** 아이콘(톱니바퀴 아이콘)을 클릭
2. 또는 키보드 단축키 사용:
   - Windows/Linux: `Ctrl + ,`
   - macOS: `Cmd + ,`

## 2단계: APIMart 모델 플랫폼 추가

### 2.1 모델 서비스 관리 액세스

설정 페이지에서:

1. 왼쪽 메뉴에서 **모델 서비스** 옵션 찾기
2. 클릭하여 모델 서비스 관리 페이지로 이동

<Frame>
  <img src="/images/integrations/cherry-studio/settings-providers.png" alt="Cherry Studio 설정 - 모델 서비스" />
</Frame>

<caption>*설정 페이지 왼쪽 메뉴에서 "모델 서비스" 옵션 찾기*</caption>

### 2.2 APIMart 공급자 추가

1. 모델 서비스 페이지 하단에서 **"+ 添加"** 버튼 클릭

<Frame>
  <img src="/images/integrations/cherry-studio/add-provider-config.png" alt="공급자 추가 대화 상자" />
</Frame>

<caption>*"+ 添加"를 클릭하면 공급자 추가 대화 상자가 열립니다*</caption>

2. **"공급자 추가"** 대화 상자에서 입력:
   - **공급자 이름**: `APIMart` (사용자 정의 가능)
   - **공급자 유형**: `OpenAI` 선택

3. **"확인"** 버튼 클릭

### 2.3 APIMart API 정보 구성

공급자를 추가한 후 오른쪽 구성 영역에서 API 정보 입력:

| 필드 | 값 |
|------|-----|
| **API 키** | APIMart API 키 (`sk-xxxxxxxxxxxx`) |
| **API 주소** | `https://api.apimart.ai` |

<Note>
**중요:**
- API 주소는 `https://api.apimart.ai`이어야 합니다 (`/v1` 또는 다른 경로 포함 안 함)
- API 키는 APIMart 콘솔에서 받은 `sk-`로 시작하는 키여야 합니다
- 입력 후 **"검测"** 버튼을 클릭하여 연결을 테스트할 수 있습니다
</Note>

## 3단계: 모델 추가 및 관리

### 3.1 모델 관리 열기

APIMart 공급자 구성 완료 후:

1. APIMart 공급자 오른쪽 상단의 스위치가 **ON**(녹색)인지 확인
2. 오른쪽 구성 영역에서 **"모델"** 섹션 찾기
3. **"관리"** 버튼을 클릭하여 모델 선택 창 열기

<Frame>
  <img src="/images/integrations/cherry-studio/provider-list.png" alt="APIMart 모델 관리" />
</Frame>

<caption>*"관리" 버튼을 클릭하여 APIMart 모델 선택 창 열기*</caption>

### 3.2 원하는 모델 추가

모델 선택 창에서:

1. 검색 상자를 사용하여 특정 모델 검색 가능
2. 상단의 카테고리 탭으로 필터링: **전체**, **추리**, **시각**, **연결**, **무료**, **임베딩**, **재순위**, **도구**
3. 원하는 모델을 찾아 오른쪽의 **+** 버튼을 클릭하여 추가:
   - `gpt-4o` / `chatgpt-4o` - OpenAI GPT-4o 모델
   - `gpt-4o-mini` - 더 빠르고 경제적인 버전
   - `claude-3` 시리즈 - Claude 3 모델 (여러 변형 포함)
   - `claude_code_sonnet-4` - Claude Code Sonnet 4
   - `claude_code_haiku-4` - Claude Code Haiku 4
   - `gemini-2.0-flash-exp` - Google Gemini 2.0 Flash

## 4단계: 대화 시작

## 자주 묻는 질문

### Q1: APIMart 서비스에 연결할 수 없나요?

**해결 방법:**

1. **Base URL 확인**:
   - Base URL이 `https://api.apimart.ai`인지 확인
   - `/v1` 접미사를 추가하지 마세요

2. **API Key 확인**:
   - API Key가 올바르고 `sk-`로 시작하는지 확인
   - [APIMart 콘솔](https://apimart.ai/console/token)에서 키가 유효한지 확인

3. **네트워크 연결 확인**:
   - `https://api.apimart.ai`에 액세스할 수 있는지 확인
   - 중국에 있는 경우 프록시 구성이 필요할 수 있습니다

### Q2: 모델 목록이 비어 있거나 새로 고칠 수 없나요?

**해결 방법:**

1. **수동으로 모델 추가**:
   - 자동 새로 고침이 실패하면 일반 모델을 수동으로 추가할 수 있습니다
   - 모델 관리에서 모델 이름을 수동으로 입력(예: `gpt-4o`)

2. **API 권한 확인**:
   - API Key에 모델 목록에 대한 액세스 권한이 있는지 확인
   - APIMart 지원팀에 문의하여 계정 상태 확인

### Q3: 대화 중 오류 메시지가 표시되나요?

**일반적인 오류 및 해결 방법:**

| 오류 메시지 | 원인 | 해결 방법 |
|-----------|------|---------|
| `401 Unauthorized` | 유효하지 않거나 만료된 API Key | API Key를 다시 받아 구성 업데이트 |
| `429 Too Many Requests` | 요청 속도 제한 초과 | 잠시 기다렸다가 재시도 |
| `500 Internal Server Error` | 임시 서버 문제 | 몇 분 기다렸다가 재시도 |
| `insufficient_quota` | 계정 잔액 부족 | 콘솔에서 충전 |

### Q4: API 사용량 및 비용을 어떻게 확인하나요?

[APIMart 콘솔](https://apimart.ai/console)에 로그인하여 확인:
- 📊 API 호출 통계
- 💰 비용 세부 정보
- 📈 사용 추세 차트

## 기능

Cherry Studio + APIMart를 사용하면 다음을 수행할 수 있습니다:

- 💬 **다중 모델 대화** - 동일한 인터페이스에서 다양한 AI 모델 사용
- 🖼️ **이미지 이해** - 이미지 입력을 지원하는 다중 모달 대화
- 📝 **컨텍스트 관리** - 대화 기록 및 컨텍스트의 지능형 관리
- 🎨 **사용자 정의 프롬프트** - 프롬프트 템플릿 생성 및 관리
- 📊 **대화 내보내기** - 대화 기록을 Markdown 등의 형식으로 내보내기
- 🔄 **모델 비교** - 여러 모델을 동시에 사용하고 출력 비교

```java Java
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String url = "https://api.apimart.ai/v1/images/generations";

        String payload = """
        {
          "model": "doubao-seedream-5-0-lite",
          "prompt": "정원에서 노는 골든 리트리버, 맑은 날, 고화질 사진 스타일",
          "size": "16:9",
          "resolution": "2K",
          "n": 1
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(url))
            .header("Authorization", "Bearer <token>")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());

        System.out.println(response.body());
    }
}
```

## Body

<ParamField body="model" type="string" default="doubao-seedream-5-0-lite" required>
  이미지 생성 모델 이름

  Example: `"doubao-seedream-5-0-lite"`
</ParamField>

<ParamField body="prompt" type="string" required>
  이미지 생성을 위한 텍스트 설명
</ParamField>

<ParamField body="size" type="string" default="1:1">
  이미지 종횡비

  지원되는 종횡비:
  - `1:1` - 정사각형 (기본값)
  - `4:3` - 가로 4:3
  - `3:4` - 세로 3:4
  - `16:9` - 가로 와이드스크린
  - `9:16` - 세로
  - `3:2` - 가로 3:2
  - `2:3` - 세로 2:3
  - `21:9` - 울트라와이드
  - `auto` - 참조 이미지 종횡비 자동 매칭 (image_urls 필요)

  > **참고:** Seedream-5.0-Lite는 `9:21` 비율을 지원하지 않습니다
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  이미지 해상도

  지원되는 해상도:
  - `2K` - 표준 해상도 (기본값)
  - `3K` - 고해상도

  > **참고:** Seedream-5.0-Lite는 2K와 3K 해상도만 지원합니다. 1K와 4K는 지원되지 않습니다

  **해상도 참조 크기:**
  | 해상도 | 1:1 크기 | 16:9 크기 |
  |--------|---------|----------|
  | 2K | 2048x2048 | 2848x1600 |
  | 3K | 3072x3072 | 4096x2304 |

  <Expandable title="전체 픽셀 매핑 테이블">
    **2K 해상도**
    | 비율 | 픽셀 |
    |------|------|
    | 1:1 | 2048×2048 |
    | 4:3 | 2304×1728 |
    | 3:4 | 1728×2304 |
    | 16:9 | 2848×1600 |
    | 9:16 | 1600×2848 |
    | 3:2 | 2496×1664 |
    | 2:3 | 1664×2496 |
    | 21:9 | 3136×1344 |

    **3K 해상도**
    | 비율 | 픽셀 |
    |------|------|
    | 1:1 | 3072×3072 |
    | 4:3 | 3456×2592 |
    | 3:4 | 2592×3456 |
    | 16:9 | 4096×2304 |
    | 9:16 | 2304×4096 |
    | 3:2 | 3744×2496 |
    | 2:3 | 2496×3744 |
    | 21:9 | 4704×2016 |
  </Expandable>
</ParamField>

<ParamField body="n" type="integer">
  생성할 이미지 수

  범위: 1-4

  기본값: 1

  **참고:**
  - `n > 1`일 때 연속 이미지 생성 모드가 자동으로 활성화됩니다
  - **숫자만 입력하세요 (예: `1`). 따옴표를 사용하면 오류가 발생합니다**

  생성 수에 따라 사전 청구됩니다
</ParamField>

<ParamField body="image_urls" type="array">
  이미지-이미지 생성을 위한 참조 이미지 URL 목록

  **제한:**
  - 이미지 형식: jpeg, png
  - 종횡비 (너비/높이) 범위: [1/3, 3]
  - 너비와 높이 (px) > 14
  - 크기: 이미지당 10MB 이하
  - 총 픽셀: 6000×6000 px 이하
</ParamField>

<ParamField body="output_format" type="string" default="jpeg">
  출력 이미지 형식

  - `jpeg`: JPEG 형식 (기본값)
  - `png`: PNG 형식, 투명 배경 등에 적합

  > **참고:** `output_format` 매개변수는 Seedream-5.0-Lite 전용입니다. 다른 이미지 모델에서는 이 매개변수가 무시됩니다.
</ParamField>

<ParamField body="sequential_image_generation" type="string" default="disabled">
  연속 이미지 생성 모드

  여러 이미지를 생성할지 여부를 제어합니다:

  - `disabled`: 연속 모드 비활성화 (기본값)
  - `auto`: 연속 모드 활성화, 여러 이미지 생성 가능

  **사용 참고:**
  - `n > 1`일 때 자동으로 `auto`로 설정됩니다
</ParamField>

<ParamField body="sequential_image_generation_options" type="object">
  연속 이미지 생성 옵션

  `sequential_image_generation`이 `auto`로 설정된 경우 사용 가능

  **속성:**
  - `max_images` (integer): 생성할 이미지 수 지정

  **예시:**
  ```json
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 4
  }
  ```
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  생성된 이미지에 워터마크를 추가할지 여부

  - `true`: 워터마크 추가
  - `false`: 워터마크 없음 (기본값)
</ParamField>

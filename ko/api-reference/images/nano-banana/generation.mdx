## 요청 본문

<ParamField body="model" type="string" default="gemini-2.5-flash-image-preview" required>
  이미지 생성 모델 이름

  Example: `"gemini-2.5-flash-image-preview"`
</ParamField>

<ParamField body="prompt" type="string" required>
  이미지 생성을 위한 텍스트 설명

  최대 1000자
</ParamField>

<ParamField body="size" type="string">
  이미지 생성 크기

  지원되는 형식:
  - 비율: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9`
</ParamField>

<ParamField body="n" type="integer">
  생성할 이미지 수

  1로 고정
  
  **⚠️ 주의：** 순수한 숫자를 입력하세요（예：`1`）。인용부호를 사용하면 오류가 발생합니다
</ParamField>

<ParamField body="image_urls" type="array">
  이미지-이미지 또는 이미지 편집을 위한 참조 이미지 URL 목록

  - 최대 5개의 이미지
  - 각 이미지는 10MB를 초과하지 않아야 합니다
  - 지원 형식: .jpeg, .jpg, .png, .webp
</ParamField>

<ParamField body="mask_url" type="string">
  마스크 이미지 URL

  - PNG 형식이어야 합니다
  - 크기는 참조 이미지와 일치해야 합니다
  - 최대 4MB
</ParamField>

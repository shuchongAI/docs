```java Java
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String url = "https://api.apimart.ai/v1/images/generations";

        String payload = """
        {
          "model": "gemini-3.1-flash-image-preview",
          "prompt": "사이버펑크 스타일의 야경 도시, 네온 불빛이 빛나는",
          "size": "16:9",
          "resolution": "2K",
          "n": 1
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(url))
            .header("Authorization", "Bearer <token>")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());

        System.out.println(response.body());
    }
}
```

## Body

<ParamField body="model" type="string" default="gemini-3.1-flash-image-preview" required>
  이미지 생성 모델 이름

  Example: `"gemini-3.1-flash-image-preview"`
</ParamField>

<ParamField body="prompt" type="string" required>
  이미지 생성을 위한 텍스트 설명
</ParamField>

<ParamField body="size" type="string">
  이미지 종횡비

  지원되는 비율:
  - `1:1` - 정사각형, 아바타, 소셜 미디어
  - `3:2` / `2:3` - 표준 사진
  - `4:3` / `3:4` - 전통적인 디스플레이 비율
  - `16:9` / `9:16` - 와이드스크린/세로 비디오 커버
  - `5:4` / `4:5` - Instagram 이미지
  - `21:9` - 울트라와이드 배너
  - `1:4` / `4:1` - 긴 포스터/배너
  - `1:8` / `8:1` - 극단적 긴 이미지/배너 광고
</ParamField>

<ParamField body="resolution" type="string" default="1K">
  출력 이미지 해상도

  지원되는 값:
  - `0.5K` - 약 512px, 저해상도 미리보기
  - `1K` - 약 1024px, 표준 해상도 (기본값)
  - `2K` - 약 2048px, 고해상도
  - `4K` - 약 4096px, 초고해상도

  > **참고:** 해상도에 따라 요금이 다릅니다. 4K는 1K보다 비용이 높습니다.
</ParamField>

<ParamField body="n" type="integer">
  생성할 이미지 수

  기본값: 1

  **⚠️ 참고:** 숫자만 입력하세요 (예: `1`). 따옴표를 사용하면 오류가 발생합니다
</ParamField>

<ParamField body="image_urls" type="array">
  이미지-이미지 생성을 위한 참조 이미지 URL 목록

  **제한:**
  - 최대 **14장** 참조 이미지 (권장: 최대 10장 객체 참조 + 4장 캐릭터 참조)
  - 이미지당 10MB 이하
  - 지원 형식: jpeg, png, webp
</ParamField>

<ParamField body="google_search" type="boolean" default="false">
  Google 텍스트 검색 강화 활성화

  - `true`: 모델이 웹 텍스트 정보를 검색하여 이미지 생성을 지원합니다
  - `false`: 비활성화 (기본값)

</ParamField>

<ParamField body="google_image_search" type="boolean" default="false">
  Google 이미지 검색 강화 활성화

  - `true`: 텍스트 검색 외에 참조 이미지도 검색하여 생성을 지원합니다
  - `false`: 비활성화 (기본값)

  > **참고:** `google_search: true`와 함께 사용해야 합니다
</ParamField>

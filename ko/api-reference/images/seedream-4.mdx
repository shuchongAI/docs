# Seedream-4.0 이미지 생성

고품질 이미지 생성을 위해 Seedream-4.0 모델을 사용합니다.

## 주요 기능

- 비동기 처리: 후속 쿼리를 위한 작업 ID 반환
- 다중 모드: 텍스트-투-이미지, 이미지-투-이미지, 이미지 편집
- 링크 만료: 생성된 이미지 링크는 24시간 동안 유효

## 요청 예시

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedream-4.0",
    "prompt": "대나무 숲에서 노는 귀여운 판다",
    "size": "1:1",
    "n": 1
  }'
```

</RequestExample>

## 매개변수

<ParamField query="model" type="string" required>
  모델 이름. "seedream-4.0"을 사용하세요.
</ParamField>

<ParamField query="prompt" type="string" required>
  이미지 생성을 위한 텍스트 설명.
</ParamField>

<ParamField query="size" type="string">
  출력 크기. 지원: "1:1", "1792x1024", "1024x1792".
</ParamField>

<ParamField query="n" type="integer">
  생성할 이미지 수. 기본값: 1.
</ParamField>

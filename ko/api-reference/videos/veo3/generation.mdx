## 인증

<ParamField header="Authorization" type="string" required>
  모든 API 엔드포인트에서 Bearer Token 인증이 필요합니다

  API Key 획득:

  [API Key 관리 페이지](https://apimart.ai/console/token)에 접속하여 API Key를 획득하세요

  사용 시 요청 헤더에 추가:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## 요청 본문

<ParamField body="model" type="string" default="veo3.1-fast" required>
  비디오 생성 모델명

  사용 가능한 모델：
  - `veo3.1-fast` - 빠른 생성 모델, 빠른 미리보기 및 반복 작업에 적합
  - `veo3.1-quality` - 고품질 생성 모델, 최종 제작에 적합

  Example: `"veo3.1-fast"`
</ParamField>

<ParamField body="prompt" type="string" required>
  비디오 생성 텍스트 설명
</ParamField>

<ParamField body="duration" type="integer">
  비디오 길이（초）

  고정값：`8`（VEO3은 8초 길이만 지원）
</ParamField>

<ParamField body="aspect_ratio" type="string">
  비디오 해상도

  지원 형식：
  - `16:9` (가로)
  - `9:16` (세로)
</ParamField>

<ParamField body="generation_type" type="string">
  비디오 생성 유형

  지원 유형:
  - `frame` - 프레임-투-비디오 (FL 모드)
  - `reference` - 참조 이미지-투-비디오
</ParamField>

<ParamField body="image_urls" type="array<url>">
  비디오 생성에 사용할 참조 이미지 URL 배열

  공개 접근 가능한 이미지 URL (http:// 또는 https://) 지원

  예시: `["https://example.com/image.jpg"]`

  **제한사항:**
  - 최대 3장
  - 프레임-투-비디오 모드의 경우: 첫 번째 이미지가 시작 프레임, 두 번째 이미지가 종료 프레임
  - 최대 파일 크기：10MB
  - 지원 형식：.jpeg、.jpg、.png、.webp
</ParamField>

```java Java
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String url = "https://api.apimart.ai/v1/videos/{video_id}/remix";

        String payload = """
        {
          "model": "sora-2",
          "prompt": "장면에 놀고 있는 강아지 추가"
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create(url))
            .header("Authorization", "Bearer <token>")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());

        System.out.println(response.body());
    }
}
```

```c C
#include <stdio.h>

int main(void) {
    CURL *curl;

    curl = curl_easy_init();

        const char *url = "https://api.apimart.ai/v1/videos/{video_id}/remix";
```

```objectivec Objective-C
int main(int argc, const char * argv[]) {

            dataTaskWithRequest:request

                    return;
                }
                NSString *result = [[NSString alloc] initWithData:data 
                                                        encoding:NSUTF8StringEncoding];
                NSLog(@"%@", result);
            }];
        
        [task resume];
        [[NSRunLoop mainRunLoop] run];
```

## 경로 매개변수

<ParamField path="video_id" type="string" required>
  원본 비디오 작업 ID

  이전 비디오 생성 요청에서 반환된 작업 ID입니다
</ParamField>

## 인증

<ParamField header="Authorization" type="string" required>
  모든 API 엔드포인트는 Bearer 토큰 인증이 필요합니다

  API 키 받기:

  [API 키 관리 페이지](https://api.apimart.ai/console/token)를 방문하여 API 키를 받으세요

  요청 헤더에 추가:

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## 본문

<ParamField body="model" type="string" default="sora-2" required>
  비디오 리믹스 모델 이름

  지원되는 모델:
  - `sora-2` - 표준 버전
  - `sora-2-pro` - 프로페셔널 버전, 더 긴 재생 시간 지원

  예시: `"sora-2"` 또는 `"sora-2-pro"`
</ParamField>

<ParamField body="prompt" type="string" required>
  리믹스 지시 설명

  비디오에 적용하려는 수정 사항을 설명하세요, 최대 1000자
</ParamField>

<ParamField body="duration" type="integer">
  비디오 재생 시간 (초)

  - `sora-2`: 10초 또는 15초 지원
  - `sora-2-pro`: 15초 (HD) 또는 25초 지원

  예시: `15`
</ParamField>

<ResponseField name="created" type="integer">
  작업 생성 타임스탬프

</ResponseField>

<ResponseField name="progress" type="integer">

</ResponseField>

<ResponseField name="status" type="string">
  작업 상태

  가능한 값:
  - `pending` - 처리 대기 중
  - `processing` - 처리 중
  - `completed` - 완료됨
  - `failed` - 실패
</ResponseField>

# Wan2.5-Video 비디오 생성

고품질 비디오 생성을 위해 Wan2.5-Video 모델을 사용합니다.

## 하이라이트

- 비동기 처리: 후속 쿼리를 위한 작업 ID 반환
- 다중 모드: 텍스트-투-비디오, 이미지-투-비디오
- 만료 링크: 생성된 비디오 링크는 24시간 동안 유효

## 요청 예시

<RequestExample>

```bash
curl -X POST https://apimart.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.5-video",
    "prompt": "밤의 도시 교통",
    "duration": 12,
    "resolution": "1920x1080"
  }'
```

</RequestExample>

## 파라미터

<ParamField query="model" type="string" required>
  모델 이름. "wan2.5-video"를 사용합니다.
</ParamField>

<ParamField query="prompt" type="string" required>
  비디오 생성을 위한 텍스트 설명.
</ParamField>

<ParamField query="duration" type="integer">
  비디오 길이 (초). 기본값: 12.
</ParamField>

<ParamField query="resolution" type="string">
  비디오 해상도. 지원: "1920x1080", "1280x720".
</ParamField>

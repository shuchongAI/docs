# VEO3 비디오 생성

고품질 비디오 생성을 위해 VEO3 모델을 사용합니다.

## 주요 기능

- 비동기 처리: 후속 쿼리를 위한 작업 ID 반환
- 다중 모드: 텍스트-투-비디오, 이미지-투-비디오
- 링크 만료: 생성된 비디오 링크는 24시간 동안 유효

## 요청 예시

<RequestExample>

```bash
curl -X POST https://api.apimart.ai/v1/videos/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1",
    "prompt": "정원에서 노는 새끼 고양이",
    "duration": 10,
    "aspect_ratio": "16:9"
  }'
```

</RequestExample>

## 매개변수

<ParamField query="model" type="string" required>
  모델 이름. "veo3"를 사용하세요.
</ParamField>

<ParamField query="prompt" type="string" required>
  비디오 생성을 위한 텍스트 설명.
</ParamField>

<ParamField query="duration" type="integer">
  비디오 길이 (초). 기본값: 10.
</ParamField>

<ParamField query="aspect_ratio" type="string">
  비디오 해상도

  지원 형식:
  - `16:9` (가로)
  - `9:16` (세로)
</ParamField>

<ParamField query="generation_type" type="string">
  비디오 생성 유형

  지원 유형:
  - `frame` - 프레임-투-비디오 (FL 모드)
  - `reference` - 참조 이미지-투-비디오

  **주의: `veo3.1-quality` 모델은 `reference` 모드를 지원하지 않습니다**
</ParamField>

<ParamField query="image_urls" type="array">
  이미지-투-비디오 생성을 위한 참조 이미지 URL 목록

  - 최대 3장
  - 프레임-투-비디오 모드의 경우: 첫 번째 이미지가 시작 프레임, 두 번째 이미지가 종료 프레임
  - 지원 형식: .jpeg, .jpg, .png, .webp
</ParamField>

<ParamField query="resolution" type="string">
  비디오 해상도

  지원 값:
  - `720p` (기본값)
  - `1080p` (16:9 비율만 지원)
</ParamField>

<ParamField query="enable_gif" type="boolean">
  GIF 출력 형식 활성화. 기본값: false.

  주의: GIF와 1080p 해상도는 동시에 사용할 수 없습니다.
</ParamField>

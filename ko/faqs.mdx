## 연결 및 사용

### Q1: APIMart 서비스에 연결할 수 없나요?

APIMart 서비스에 연결할 수 없는 경우 다음을 확인하세요:

- API Key가 올바르게 구성되었는지 확인
- 네트워크 연결이 정상인지 확인
- API 엔드포인트 주소가 올바른지 확인: `https://api.apimart.ai`
- 방화벽 또는 프록시 설정이 연결을 차단하지 않는지 확인

### Q2: 모델 선택 후 응답이 없거나 오류가 발생하나요?

다음과 같은 이유일 수 있습니다:

- API Key가 올바르게 구성되지 않았거나 만료됨
- 선택한 모델이 일시적으로 사용할 수 없음
- 요청 형식이 올바르지 않음
- 계정 잔액 부족

[APIMart 대시보드](https://api.apimart.ai)에서 계정 상태와 API 키를 확인하세요.

### Q3: 대화 오류나 중단이 발생하면 어떻게 하나요?

대화 오류가 발생한 경우:

- 오류 메시지의 구체적인 정보 확인
- 요청 매개변수가 API 사양을 준수하는지 확인
- API 문서에서 올바른 요청 형식 확인
- 문제가 지속되면 기술 지원에 문의

## 계정 관리

### Q4: API 사용량과 비용을 어떻게 확인하나요?

다음 방법으로 사용량과 비용을 확인할 수 있습니다:

1. [APIMart 대시보드](https://api.apimart.ai)에 로그인
2. 콘솔에서 사용 통계 확인
3. 상세한 호출 기록 및 요금 명세 확인

### Q5: 기술 지원을 받으려면 어떻게 하나요?

기술 지원이 필요한 경우 다음 방법으로 문의하세요:

- [Discord 커뮤니티](https://discord.gg/V8zqssyZ5c) 참여
- [Twitter @APIMart_](https://x.com/APIMart_) 팔로우
- WeChat 고객 서비스 추가 (대시보드 QR 코드 스캔)
- 기술 지원 팀에 이메일 전송

## 보안 및 구성

### Q6: API Key를 관리하고 보호하는 방법은?

API Key 보안을 보호하세요:

- 공개 코드 저장소에 API Key를 커밋하지 마세요
- 환경 변수를 사용하여 API Key 저장
- 정기적으로 API Key 교체
- API Key가 유출되면 대시보드에서 즉시 재생성

## 기능 및 특징

### Q7: 어떤 AI 모델이 지원되나요?

APIMart는 다양한 주요 AI 모델을 지원합니다:

- **텍스트 모델**: GPT-5, Claude, Gemini 등
- **이미지 모델**: GPT-4o-image, Gemini-2.5-Flash-Image 등
- **비디오 모델**: Sora2, VEO3 등
- **오디오 모델**: Whisper-1, TTS 등

지원되는 특정 모델은 [API 문서](/ko/index)를 확인하세요.

### Q8: OpenAI API와의 호환성은 어떤가요?

APIMart는 OpenAI API 형식과 완벽하게 호환됩니다:

- API 엔드포인트를 직접 교체 가능
- 동일한 요청 형식 사용
- 기존 코드를 수정할 필요 없음
- 모든 주요 OpenAI API 기능 지원

## 비용 최적화

### Q9: API 호출 비용을 최적화하는 방법은?

비용 최적화를 위한 제안:

- 필요에 맞는 모델 선택 (모델마다 가격이 다름)
- 스트리밍 응답을 사용하여 대기 시간 단축
- 합리적인 토큰 제한 설정
- 일반적인 응답 캐싱
- 사용량 모니터링하여 예상치 못한 소비 방지

## APIMart API 플랫폼

APIMart는 통합 AI 모델 API 인터페이스를 제공하여 최신 AI 모델을 쉽게 호출할 수 있습니다.

</Card>

## 시작하기

<Steps>
  <Step title="API 키 받기">
    [API 키 관리 페이지](https://api.apimart.ai/console/token)를 방문하여 키를 발급받으세요
  </Step>
  <Step title="모델 선택">
    문서를 둘러보고 필요에 맞는 AI 모델을 선택하세요
  </Step>
  <Step title="API 호출">
    선호하는 프로그래밍 언어를 사용하여 API를 호출하세요
  </Step>
</Steps>

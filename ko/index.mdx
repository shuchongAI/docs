## APIMart - 비용 효율적인 엔터프라이즈급 AI API 플랫폼

APIMart는 GPT-5, Claude, Gemini와 같은 최고 수준의 AI 모델에 대한 액세스를 제공하는 선도적인 통합 AI 모델 API 플랫폼입니다. 개발자와 기업에게 간단하고 안정적이며 고성능이면서 비용 효율적인 AI API 서비스를 제공하여 텍스트 생성, 이미지 생성, 비디오 생성 등을 지원합니다.

## 지원되는 AI 모델

APIMart 플랫폼은 시장에서 가장 발전된 AI 모델에 대한 액세스를 제공합니다. 모든 모델은 통합된 API 인터페이스를 통해 액세스할 수 있으며 OpenAI API 표준과 호환됩니다.

### 텍스트 생성 모델 (LLM)

채팅, 텍스트 완성, 코드 생성 등을 위한 세계 최고의 대규모 언어 모델에 액세스:

- **GPT-5**: 향상된 추론 기능을 갖춘 OpenAI의 최신 플래그십 모델
- **GPT-4o & GPT-4o Mini**: 성능과 비용의 균형을 맞춘 OpenAI의 멀티모달 모델
- **Claude Sonnet 4.5 & Haiku 4.5**: 복잡한 추론을 위한 Anthropic의 고급 AI 모델
- **Gemini 2.0 Flash & Flash Thinking**: Google의 최신 멀티모달 AI 모델

### 이미지 생성 모델

텍스트 설명에서 고품질 AI 이미지 생성:

- **GPT-4o-image**: OpenAI의 고급 이미지 생성 모델
- **Nano banana (Gemini 2.5 Flash Image)**: Google의 효율적인 이미지 생성 모델

### 비디오 생성 모델

최첨단 AI 비디오 생성 기술:

- **OpenAI Sora2**: OpenAI의 최신 비디오 생성 모델
- **Google VEO3**: Google의 고급 비디오 생성 모델

### 오디오 모델

음성 인식 및 텍스트 음성 변환 기능:

- **Whisper-1**: OpenAI의 음성-텍스트 변환 모델
- **TTS**: 고품질 텍스트 음성 변환 생성

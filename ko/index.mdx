## OpenAI 호환 엔드포인트

## 투명한 가격

구독 없이 종량제. 공식 프로바이더 요금보다 저렴하면서 동일한 품질과 저지연 유지.

- **투명한 가격**: 모든 모델의 명확한 토큰당 요금
- **대량 할인**: 엔터프라이즈 사용량에 대한 낮은 요금
- **숨겨진 수수료 없음**: 표시된 가격 그대로
- **종량제**: 실제 사용량만 지불

## 멀티 프로바이더 라우팅 및 엔터프라이즈 SLA

## SDK 및 코드 예제

Python, Node.js, Java용 공식 SDK. 모든 SDK가 OpenAI 호환이며 Base URL 설정 변경만으로 원활하게 마이그레이션.

<Columns cols={3}>
  <Card title="Python SDK" icon="python">
    `pip install openai`로 설치. `base_url="https://api.apimart.ai/v1"` 설정 후 기존 OpenAI 코드를 변경 없이 사용.
  </Card>
  <Card title="Node.js SDK" icon="node-js">
    `npm install openai`로 설치. `baseURL` 파라미터를 APIMart 엔드포인트로 설정하고 기존 통합 유지.
  </Card>
  <Card title="Java SDK" icon="java">
    OpenAI 호환 Java 클라이언트. 간단한 설정 변경으로 모든 APIMart 모델 및 프로바이더 액세스 가능.
  </Card>
</Columns>

## OpenAI에서 마이그레이션

5분 이내에 APIMart로 전환. 코드 재작성 불필요, Base URL과 API Key만 변경.

<Steps>
  <Step title="API Key 받기">
    APIMart에 가입하고 <a href="https://api.apimart.ai/console/token" rel="nofollow" target="_blank">콘솔 대시보드</a>에서 API Key를 받으세요.
  </Step>
  <Step title="Base URL 업데이트">
    OpenAI Base URL을 `https://api.openai.com/v1`에서 `https://api.apimart.ai/v1`로 변경. 기존 SDK 통합은 수정 없이 작동.
  </Step>
  <Step title="엔드포인트 테스트">
    새 Base URL로 기존 코드 실행. 모든 OpenAI 호환 엔드포인트는 동일한 형식으로 응답. 콘솔 대시보드에서 지연 시간 및 속도 제한 확인.
  </Step>
</Steps>


## 통합 엔드포인트로 지원되는 모델

하나의 OpenAI 호환 게이트웨이로 주요 LLM 프로바이더에 액세스. 모든 모델은 `/v1/chat/completions` 엔드포인트에서 사용 가능, 일관된 요청/응답 형식.

### 채팅 완성 모델

채팅, 완성, 코드 작업용 주요 언어 모델:

- **GPT-5**: 향상된 추론 기능을 갖춘 OpenAI 플래그십 모델
- **GPT-4o & GPT-4o Mini**: 성능과 비용의 균형을 맞춘 멀티모달 모델
- **Claude Sonnet 4.5 & Haiku 4.5**: 복잡한 추론을 위한 Anthropic 모델
- **Gemini 2.0 Flash & Flash Thinking**: Google 멀티모달 모델

### 이미지 생성 모델

텍스트 프롬프트에서 `/v1/images/generations` 엔드포인트를 통해 이미지 생성:

- **GPT-4o Image**: OpenAI 이미지 생성 모델
- **Gemini 2.5 Flash Image**: Google 효율적 이미지 모델

### 비디오 생성 모델

비동기 작업 추적 및 Webhook 콜백이 있는 비디오 생성:

- **OpenAI Sora2**: OpenAI 비디오 생성 모델
- **Google VEO3**: Google 비디오 생성 모델

### 음성 모델

OpenAI 호환 엔드포인트를 통한 음성-텍스트 및 텍스트-음성 변환:

- **Whisper-1**: OpenAI 전사 모델
- **TTS**: 여러 음성으로 텍스트 음성 변환

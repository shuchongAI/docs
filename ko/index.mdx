## 플랫폼 이점

### 투명한 가격

구독 없이 종량제. 모든 모델의 명확한 토큰당 요금, 공식 프로바이더보다 저렴하면서 동일한 품질과 저지연 유지.

- 모든 모델의 명확한 토큰당 요금
- 엔터프라이즈 사용량 대량 할인
- 숨겨진 수수료 없음
- 실제 사용량만 지불

### 멀티 프로바이더 라우팅 및 엔터프라이즈 SLA

여러 LLM 프로바이더 간 인텔리전트 라우팅 내장으로 높은 가동 시간과 저지연 보장. 한 프로바이더에 문제가 발생하면 요청이 자동으로 백업 프로바이더로 라우팅되어 서비스 중단 없음.

### 99.9% 가동 시간 SLA

자동 장애 조치 기능이 있는 엔터프라이즈급 안정성. 멀티 프로바이더 라우팅으로 최소 지연 시간으로 요청 처리 보장.

### 글로벌 CDN 가속

전 세계 엣지 로케이션으로 최저 지연 시간 실현. 최적화된 라우팅으로 응답 시간 단축 및 글로벌 사용자 경험 향상.

### 속도 제한 관리

프로바이더 간 자동 속도 제한 처리. 인텔리전트 요청 분배로 스로틀링 방지 및 원활한 운영 보장.

### 실시간 상태

엔드포인트 상태 및 성능 모니터링. 이미지 및 비디오 생성의 비동기 작업 진행 상황 추적, Webhook 알림 지원. [상태 확인 →](/ko/api-reference/tasks/status)

## SDK 및 코드 예제

Python, Node.js, Java용 공식 SDK. 모든 SDK가 OpenAI 호환이며 Base URL 설정 변경만으로 원활하게 마이그레이션.

### Python SDK

`pip install openai`로 설치. Base URL을 설정하고 기존 OpenAI 코드를 변경 없이 사용.

```python
from openai import OpenAI
client = OpenAI(
    base_url="https://api.apimart.ai/v1",
    api_key="your-apimart-key"
)
```

### Node.js SDK

`npm install openai`로 설치. baseURL 파라미터를 APIMart 엔드포인트로 설정하고 기존 통합 유지.

```javascript
import OpenAI from 'openai';
const client = new OpenAI({
  baseURL: 'https://api.apimart.ai/v1',
  apiKey: 'your-apimart-key'
});
```

### Java SDK

OpenAI 호환 Java 클라이언트. 간단한 설정 변경으로 모든 APIMart 모델 및 프로바이더 액세스 가능.

## OpenAI에서 마이그레이션

5분 이내에 APIMart로 전환. 코드 재작성 불필요——Base URL과 API Key만 변경.

<Steps>
  <Step title="API Key 받기">
    APIMart에 가입하고 <a href="https://api.apimart.ai/console/token" rel="nofollow" target="_blank">콘솔 대시보드</a>에서 API Key를 받으세요.
  </Step>
  <Step title="Base URL 업데이트">
    OpenAI Base URL을 APIMart 엔드포인트로 변경. 기존 SDK 통합은 수정 없이 작동.
  </Step>
  <Step title="엔드포인트 테스트">
    새 Base URL로 기존 코드 실행. 모든 OpenAI 호환 엔드포인트는 동일한 형식으로 응답. 콘솔 대시보드에서 지연 시간 및 속도 제한 확인.
  </Step>
</Steps>
